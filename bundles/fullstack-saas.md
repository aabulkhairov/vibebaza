## Для кого эта связка

Для разработчиков и стартаперов, которые хотят быстро запустить SaaS-продукт. Связка покрывает весь цикл разработки — от базы данных до мониторинга в продакшене.

## Что включено

### MCP-серверы

**PostgreSQL** — надежная реляционная база данных. Claude сможет напрямую работать с вашими таблицами, писать миграции и оптимизировать запросы.

**AWS** — облачная инфраструктура. Деплой на EC2, хранение файлов в S3, управление секретами через Secrets Manager.

**GitLab** — CI/CD и Git-репозитории. Автоматические пайплайны, код-ревью, управление релизами.

**Sentry** — мониторинг ошибок. Отслеживание исключений, производительности и пользовательских сессий.

### Навыки

**Rails Developer** — генерация Rails-кода по лучшим практикам. Модели, контроллеры, сервисы, тесты.

**Python Developer** — FastAPI эндпоинты, асинхронная обработка, интеграция с ML-моделями.

**Frontend Design** — создание современных интерфейсов с Tailwind CSS. Адаптивный дизайн, темная тема, анимации.

### Агенты

**Research Agent** — исследование архитектурных решений, поиск библиотек, анализ конкурентов.

**Test Generator** — автоматическая генерация unit-тестов и интеграционных тестов.

## Как использовать

1. **Подключите MCP-серверы** к вашему Claude клиенту
2. **Активируйте навыки** в настройках
3. **Опишите задачу** — Claude подберет нужные инструменты

### Пример промпта

```
Создай Rails API для управления подписками пользователей:
- Модель Subscription с полями plan, status, expires_at
- CRUD эндпоинты с аутентификацией
- Интеграция с Stripe для обработки платежей
- Миграции и тесты
```

## Результат

- Рабочий backend с API и базой данных
- CI/CD пайплайн для автоматического деплоя
- Мониторинг ошибок в реальном времени
- Покрытие тестами основного функционала
