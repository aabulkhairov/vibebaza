## Для кого эта связка

Для владельцев интернет-магазинов, маркетплейс-селлеров и e-commerce менеджеров, которые хотят автоматизировать рутину и масштабировать продажи.

## Что включено

### MCP-серверы

**Airtable** — база данных товаров и заказов. Управление каталогом, отслеживание статусов, автоматизация воркфлоу.

**Notion** — документация и процессы. Инструкции для команды, FAQ, база знаний по продуктам.

**Apify** — парсинг конкурентов и маркетплейсов. Мониторинг цен, сбор отзывов, анализ ассортимента.

**Browserbase** — автоматизация браузера. Публикация товаров, обновление цен, выгрузка на площадки.

### Навыки

**E-commerce Marketing** — стратегии продвижения интернет-магазина. Акции, программы лояльности, ремаркетинг.

**Copywriting** — продающие тексты для карточек товаров. Заголовки, описания, УТП.

**Customer Success** — удержание клиентов и повышение LTV. Онбординг, реактивация, апсейлы.

### Агенты

**Customer Feedback Agent** — анализ отзывов и обратной связи. Выявление проблем, тренды в пожеланиях.

**Customer Onboarding Agent** — приветственные цепочки для новых клиентов. Персонализированные рекомендации.

### Промпты

**Product Description** — карточка товара с SEO-оптимизацией и продающими элементами.

**Email Subject Lines** — цепляющие заголовки для email-рассылок.

**Upsell Email** — письма с рекомендациями дополнительных товаров.

## Как использовать

1. **Импортируйте каталог** в Airtable
2. **Настройте мониторинг** конкурентов через Apify
3. **Генерируйте описания** с помощью Copywriting навыка
4. **Автоматизируйте коммуникации** с Customer Success

### Пример промпта

```
Создай карточку товара для маркетплейса:
- Название: Беспроводные наушники TWS-500
- Характеристики: Bluetooth 5.3, 30 часов работы, шумоподавление
- Целевая аудитория: молодые профессионалы
- Стиль: premium, но доступный
```

## Воркфлоу автоматизации

```
Новый товар → Описание → SEO → Публикация → Мониторинг → Оптимизация
      ↓           ↓        ↓         ↓            ↓            ↓
  Airtable   Copywriting  AI    Browserbase    Apify      Analytics
```

## Результат

- Автоматическая генерация карточек товаров
- Мониторинг цен конкурентов в реальном времени
- Персонализированные email-цепочки
- Система сбора и анализа отзывов
