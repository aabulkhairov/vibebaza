## Для кого эта связка

Для контент-менеджеров, маркетологов и владельцев бизнеса, которые хотят масштабировать производство контента без потери качества.

## Что включено

### MCP-серверы

**Notion** — центр управления контентом. Хранение идей, редакционный календарь, база знаний для команды.

**Firecrawl** — парсинг и анализ веб-страниц. Исследование конкурентов, сбор данных для статей.

**Brave Search** — поиск актуальной информации. Исследование трендов, проверка фактов, поиск источников.

**Google Drive** — хранение и совместная работа над документами. Черновики, финальные версии, архив.

### Навыки

**Content Marketing** — стратегическое планирование контента. Определение тем, форматов и каналов распространения.

**SEO Content Agent** — оптимизация для поисковых систем. Ключевые слова, мета-теги, структура статей.

**Copywriting** — написание убедительных текстов. Заголовки, CTA, описания продуктов.

### Агенты

**SEO Content Agent** — автономный агент для создания SEO-оптимизированного контента от исследования до публикации.

**Social Media Agent** — адаптация контента под разные платформы. Посты для LinkedIn, Twitter, Instagram.

**Content Calendar Agent** — планирование публикаций. Оптимальное время, частота, тематические кластеры.

### Промпты

**Blog Post Outline** — структура статьи с заголовками и ключевыми тезисами.

**Twitter Thread** — серия связанных твитов для максимального охвата.

**LinkedIn Post Hook** — цепляющее начало для постов в LinkedIn.

## Как использовать

1. **Исследуйте тему** с помощью Brave Search и Firecrawl
2. **Создайте план** используя Content Calendar Agent
3. **Напишите контент** с SEO-оптимизацией
4. **Адаптируйте** под разные платформы
5. **Опубликуйте** и отслеживайте результаты

### Пример промпта

```
Создай контент-план на месяц для B2B SaaS-продукта:
- 4 статьи в блог с SEO-оптимизацией
- 12 постов для LinkedIn
- 8 Twitter-тредов
- Темы: productivity, automation, AI tools
```

## Результат

- Полный контент-план на месяц
- SEO-оптимизированные статьи
- Адаптированный контент для соцсетей
- Аналитика эффективности публикаций
