## Для кого эта связка

Для менеджеров по продажам, SDR и руководителей отделов продаж, которые хотят автоматизировать рутину и увеличить конверсию.

## Что включено

### MCP-серверы

**Notion** — CRM и база знаний. Трекинг сделок, история коммуникаций, материалы для продаж.

**Google Drive** — хранение презентаций, коммерческих предложений и контрактов.

**Brave Search** — исследование потенциальных клиентов и компаний.

### Навыки

**Sales Development Rep** — генерация и квалификация лидов.

**Enterprise Sales** — работа с крупными клиентами и длинными циклами продаж.

**Sales Negotiation** — техники переговоров и закрытия сделок.

**Account Executive** — полный цикл продажи от первого контакта до подписания.

### Агенты

**Sales Outreach Agent** — персонализированные письма и последовательности.

**Sales Script Agent** — скрипты для звонков и встреч.

**Sales Proposal Agent** — коммерческие предложения под клиента.

**Lead Nurture Agent** — прогрев лидов контентом.

## Как использовать

1. **Исследуйте лида** с Brave Search
2. **Создайте персонализированный outreach** с Sales Outreach Agent
3. **Подготовьте скрипт встречи** с Sales Script Agent
4. **Сформируйте предложение** с Sales Proposal Agent
5. **Закройте сделку** используя навыки переговоров

### Пример промпта

```
Напиши последовательность из 5 писем для:
- Компания: SaaS в сфере HR-tech
- ЛПР: VP of Engineering
- Боль: долгий найм разработчиков
- Наш продукт: AI-рекрутинг платформа
```
