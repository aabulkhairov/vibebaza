## Для кого эта связка

Для аналитиков, продакт-менеджеров и владельцев бизнеса, которые хотят принимать решения на основе данных.

## Что включено

### MCP-серверы

**PostgreSQL** — основное хранилище данных. Транзакционная база для продуктовых метрик.

**ClickHouse** — колоночная база для аналитики. Быстрые агрегации на больших объемах данных.

**Grafana** — визуализация и мониторинг. Дашборды в реальном времени, алерты, отчеты.

**SQLite** — локальная аналитика. Быстрые эксперименты, прототипы, ad-hoc запросы.

### Навыки

**Marketing Analytics** — анализ маркетинговых кампаний. ROI, конверсии, атрибуция каналов.

**Python Developer** — скрипты для ETL, обработка данных с pandas, интеграция с API.

### Агенты

**Research Agent** — глубокий анализ данных. Поиск инсайтов, выявление аномалий, подготовка отчетов.

## Как использовать

1. **Подключите базы данных** через MCP
2. **Опишите бизнес-вопрос** — Claude напишет SQL
3. **Создайте визуализацию** в Grafana
4. **Настройте автоматические отчеты**

### Пример промпта

```
Проанализируй retention пользователей за последние 3 месяца:
- Когортный анализ по неделям регистрации
- Разбивка по источникам трафика
- Сравнение платных и бесплатных пользователей
- Визуализация в виде heatmap
```

## Пайплайн анализа

```
Сырые данные → ETL → Хранилище → Анализ → Визуализация → Инсайты
     ↓           ↓        ↓          ↓           ↓           ↓
  Логи API   Python   ClickHouse   SQL      Grafana    Research
  События    pandas   PostgreSQL   Claude   Дашборды    Agent
```

## Результат

- Автоматизированный сбор и обработка данных
- Интерактивные дашборды для команды
- SQL-запросы для любых ad-hoc вопросов
- Регулярные отчеты с инсайтами
