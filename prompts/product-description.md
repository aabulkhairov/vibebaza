# Product Description Generator

Write product descriptions that inform and convert.

## Prompt Template

```
Create a product description for:

**Product Name:** [Name]
**Product Category:** [Type of product]
**Target Customer:** [Who buys this]
**Key Features:** [3-5 main features]
**Key Benefits:** [What customers gain]
**Price Point:** [Budget/Mid/Premium]
**Competitive Advantage:** [What makes it unique]
**Use Cases:** [How customers use it]

Generate:
1. Short description (50 words)
2. Medium description (150 words)
3. Full description (300 words)
4. Bullet point features (5-7)
5. SEO meta description
```

## Description Frameworks

### FAB (Feature-Advantage-Benefit)
**Feature:** What it is/has
**Advantage:** Why it's better
**Benefit:** What customer gains

Example:
"Includes AI-powered analytics (feature) that automatically surface insights (advantage), so you can make data-driven decisions without being a data scientist (benefit)."

### Problem-Solution-Result
"Tired of [problem]? [Product] [solution]. Get [result]."

### Story-Based
"[Product] was born from [origin story]. We designed it for [audience] who need [outcome]. Every [feature] exists because [reason]."

## E-commerce Structure

1. **Hook headline**
2. **Benefit-first opening**
3. **Key features (bullets)**
4. **Social proof snippet**
5. **Use case scenario**
6. **Call to action**

## B2B SaaS Structure

1. **Value proposition headline**
2. **Problem statement**
3. **Solution overview**
4. **Feature breakdown with benefits**
5. **Integration/compatibility**
6. **Security/compliance (if relevant)**
7. **CTA with offer**

## Tone Guidelines

| Price Point | Tone | Focus |
|-------------|------|-------|
| Budget | Practical, value | Price, functionality |
| Mid-range | Balanced, helpful | Quality, features |
| Premium | Sophisticated, exclusive | Experience, prestige |
