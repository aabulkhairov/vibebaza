# Thank You Page Copy

Turn thank you pages into engagement opportunities.

## Prompt Template

```
Create thank you page copy for:

**Conversion Action:** [Form submit/Purchase/Download/Registration]
**What They Get:** [Deliverable/Confirmation]
**Next Step Goal:** [What you want them to do next]

**Context:**
- Product/Offer: [What they signed up for]
- Delivery Method: [Immediate download/Email/Access granted]
- Audience: [Who converted]

Generate:
1. Confirmation headline and message
2. Delivery instructions
3. Next step CTA
4. Secondary engagement options
5. Social sharing prompt
```

## Thank You Page Elements

### 1. Confirmation Message
- Clear confirmation of action completed
- Set expectations for next steps
- Express genuine appreciation

### 2. Delivery Instructions
- How/when they'll receive what they requested
- Check spam folder reminder (for emails)
- Access instructions (for portals/software)

### 3. Primary Next Step
- Single most valuable action
- Related to their interest
- Clear value proposition

### 4. Secondary Options
- Social sharing
- Additional resources
- Community join
- Calendar scheduling
