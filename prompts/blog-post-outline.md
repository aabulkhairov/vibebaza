# Blog Post Outline Generator

Create structured blog post outlines optimized for readers and search engines.

## Prompt Template

```
Create a comprehensive blog post outline for:

**Topic:** [Main topic/title idea]
**Target Keyword:** [Primary SEO keyword]
**Secondary Keywords:** [2-3 related terms]
**Target Audience:** [Who will read this]
**Search Intent:** [Informational/Commercial/Transactional]
**Word Count Target:** [1000/1500/2000/2500+]
**Content Goal:** [Educate/Convert/Rank/Thought leadership]

Generate:
1. SEO-optimized title (3 options)
2. Meta description
3. Full outline with H2 and H3 headers
4. Key points for each section
5. Internal linking opportunities
6. CTA placement recommendations
```

## Outline Structure

### Introduction (10%)
- Hook statement
- Problem/opportunity identification
- What reader will learn
- Why it matters now

### Body Sections (75%)
- H2: Main section headers (4-7)
  - H3: Supporting subsections
  - Key points (bullet format)
  - Examples/evidence needed
  - Visuals suggestions

### Conclusion (15%)
- Summary of key takeaways
- Next steps for reader
- Call-to-action
- Related content links

## Example Outline

**Title:** "How to Create a Sales Enablement Strategy That Actually Works"

**H2: What Is Sales Enablement (And Why It Matters)**
- H3: Definition and scope
- H3: The business impact (stats)
- Key stat: "Companies with sales enablement see 15% higher win rates"

**H2: 5 Core Components of Effective Sales Enablement**
- H3: Content and collateral
- H3: Training and coaching
- H3: Tools and technology
- H3: Process optimization
- H3: Performance analytics

[Continue pattern...]

## SEO Checklist

- [ ] Keyword in title
- [ ] Keyword in first 100 words
- [ ] Keyword in at least one H2
- [ ] Related keywords throughout
- [ ] Internal links (3-5)
- [ ] External authority links (2-3)
