# Meta Ad Copy Generator

Create thumb-stopping ad copy optimized for Facebook and Instagram.

## Prompt Template

```
Create Meta ad copy for:

**Product/Offer:** [What you're promoting]
**Target Audience:** [Demographics, interests, pain points]
**Campaign Objective:** [Awareness/Traffic/Leads/Conversions/Sales]
**Offer/Hook:** [Discount, free trial, value proposition]
**Landing Page:** [Where the ad leads]

Generate for each format:
1. Single Image Ad (3 variations)
2. Carousel Ad (4-6 cards)
3. Video Ad script (15-30 seconds)

Include:
- Primary text (125 chars visible, full version)
- Headline (40 chars)
- Description (30 chars)
- CTA button recommendation
```

## Ad Copy Frameworks

### PAS (Problem-Agitate-Solution)
**Primary Text:** "Tired of [problem]? You're not alone. [Agitate with statistics or emotion]. That's why we created [solution]. [Benefit + CTA]"

### Social Proof Lead
**Primary Text:** "Join 10,000+ [target audience] who [achieved result]. [Customer quote]. See why they switched â†’ [CTA]"

### Question Hook
**Primary Text:** "What if you could [desirable outcome] in [timeframe]? [Social proof]. Try it free for 14 days."

### Direct Benefit
**Primary Text:** "[Specific benefit] without [common pain point]. [Product name] helps you [outcome]. Get started today."

## Character Limits

| Element | Limit | Visible |
|---------|-------|---------|
| Primary Text | 125 chars | Before "See More" |
| Headline | 40 chars | Full |
| Description | 30 chars | Full |
| Link Description | 30 chars | Full |

## Best Practices

- Lead with the benefit, not the feature
- Use numbers and specifics
- Include social proof when possible
- Match ad copy to landing page
- Test multiple angles
