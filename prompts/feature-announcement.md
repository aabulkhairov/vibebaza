# Feature Announcement Copy

Write feature announcements that excite users and drive adoption.

## Prompt Template

```
Create feature announcement content for:

**Feature:**
- Name: [Feature name]
- Description: [What it does]
- User Benefit: [Why it matters]
- Availability: [All users/Plan tier/Beta]

**Context:**
- Customer Request: [Was this requested?]
- Problem Solved: [What pain point]
- Use Cases: [How customers will use it]
- Release Type: [Major/Minor/Update]

Generate:
1. Email announcement
2. In-app notification (short)
3. Blog post outline
4. Social media posts (Twitter, LinkedIn)
5. Product changelog entry
6. Help doc introduction
```

## Announcement Frameworks

### AIDA Structure
- **Attention:** What's new
- **Interest:** Why it matters to them
- **Desire:** Specific benefits/use cases
- **Action:** How to try it

### Problem-Solution-Result
- "You asked for better [X]"
- "Today we're launching [Feature]"
- "Now you can [outcome]"
