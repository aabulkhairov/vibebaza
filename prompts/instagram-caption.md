# Instagram Caption Generator

Write captions that stop the scroll and drive engagement.

## Prompt Template

```
Create Instagram captions for:

**Post Type:** [Single image/Carousel/Reel/Story]
**Content Topic:** [What the post is about]
**Image/Video Description:** [What's shown]
**Brand Voice:** [Professional/Casual/Fun/Inspirational]
**Goal:** [Engagement/Traffic/Sales/Brand awareness]
**Target Audience:** [Who you're reaching]

Generate:
1. Caption with hook, body, CTA
2. Hashtag set (20-30 relevant tags)
3. Alternative caption versions (3)
4. Story caption version
```

## Caption Structure

### The Hook (First Line)
**Question:** "Ever wonder why [common thing]?"
**Bold Statement:** "This changed everything."
**Story Start:** "Last week, something happened..."
**Challenge:** "Most people get this wrong:"
**Number:** "5 things I wish I knew earlier:"

### The Body
- Expand on the hook
- Provide value/insight
- Include personality
- Keep paragraphs short
- Use line breaks

### The CTA
- Ask a question
- Request action
- Drive to link
- Encourage saves/shares
