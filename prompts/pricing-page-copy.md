# Pricing Page Copy

Write pricing page content that converts visitors into customers.

## Prompt Template

```
Create pricing page copy for:

**Product:** [What you sell]
**Pricing Model:** [Per user/Flat rate/Usage-based/Tiered]
**Tiers:**
- Tier 1: [Name, price, key features]
- Tier 2: [Name, price, key features]
- Tier 3: [Name, price, key features]

**Target Customers:** [Who each tier serves]
**Recommended Tier:** [Which to highlight]
**Money-Back Guarantee:** [Yes/No, duration]
**Free Trial:** [Yes/No, duration]

Generate:
1. Page headline and subhead
2. Tier names and taglines
3. Feature list copy for each tier
4. "Best for" descriptions
5. FAQ section (5-7 questions)
6. Trust elements copy
```

## Pricing Page Structure

### Above the Fold
- Clear headline (value-focused)
- Brief subhead (reduce friction)
- Toggle (monthly/annual if applicable)
- Tier cards with CTAs

### Tier Card Elements
- Tier name
- Price (with billing period)
- Short description
- Feature list
- CTA button
- "Best for" statement

### Below Tiers
- Feature comparison table
- Social proof
- FAQ section
- Final CTA
