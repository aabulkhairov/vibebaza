Омнимодальный фреймворк AIGC, который легко конвертирует рабочие процессы ComfyUI в MCP инструменты без написания кода. Поддерживает генерацию текста, изображений, звука и видео с использованием локального ComfyUI или облачного режима RunningHub.

## Установка

### UVX (временно)

```bash
uvx pixelle@latest
```

### Установка через Pip

```bash
pip install -U pixelle
pixelle
```

### Из исходного кода

```bash
git clone https://github.com/AIDC-AI/Pixelle-MCP.git
cd Pixelle-MCP
uv run pixelle
```

### Docker

```bash
git clone https://github.com/AIDC-AI/Pixelle-MCP.git
cd Pixelle-MCP
cp .env.example .env
docker compose up -d
```

## Возможности

- Поддержка всех модальностей: Поддерживает полномодальную конвертацию и генерацию TISV (текст, изображения, звук/речь, видео)
- Два режима выполнения: Локальная самохостинговая среда ComfyUI + облачный сервис RunningHub ComfyUI
- Экосистема ComfyUI: Построен на ComfyUI, наследуя все возможности открытой экосистемы ComfyUI
- Разработка без кода: Решение "рабочий процесс как MCP инструмент", обеспечивающее разработку без кода и динамическое добавление новых MCP инструментов
- MCP сервер: Основан на протоколе MCP, поддерживает интеграцию с любым MCP клиентом (Cursor, Claude Desktop и т.д.)
- Веб-интерфейс: Фреймворк на основе Chainlit с элементами управления UI и поддержкой интеграции других MCP серверов
- Деплой в один клик: Поддерживает установку через PyPI, команды CLI, методы развертывания Docker
- Упрощенная конфигурация: Использует схему конфигурации через переменные окружения
- Поддержка нескольких LLM: Поддерживает OpenAI, Ollama, Gemini, DeepSeek, Claude, Qwen и другие

## Переменные окружения

### Опциональные
- `PORT` - Пользовательский порт для сервиса (по умолчанию: 9004)

## Примечания

Система автоматически запускает мастер конфигурации при первом запуске, который поможет выбрать движок выполнения (ComfyUI/RunningHub) и настроить LLM. Веб-интерфейс доступен по адресу http://localhost:9004, а MCP эндпоинт — по адресу http://localhost:9004/pixelle/mcp. Логин по умолчанию: username: dev, password: dev.
