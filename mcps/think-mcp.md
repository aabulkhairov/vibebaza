MCP сервер, который предоставляет инструмент 'think' для структурированных рассуждений в агентских AI воркфлоу, позволяя AI агентам делать паузы и записывать явные мысли во время сложных рассуждений или многошагового использования инструментов.

## Установка

### UVX

```bash
uvx think-mcp
```

## Конфигурация

### Базовая конфигурация

```json
"mcpServers": {
    "think-mcp": {
        "command": "uvx",
        "args": ["think-mcp"],
        "enabled": true
    }
}
```

### Конфигурация расширенного режима

```json
"mcpServers": {
    "think-mcp": {
        "command": "uvx",
        "args": ["think-mcp", "--advanced"],
        "enabled": true,
        "env": {
            "TAVILY_API_KEY": "... YOUR TAVILY API KEY HERE ..."
        }
    }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `think` | Добавляет мысль в лог для структурированных рассуждений без изменения окружения или базы данных |
| `criticize` | Дополнительный инструмент, доступный в расширенном режиме |
| `plan` | Дополнительный инструмент, доступный в расширенном режиме |
| `search` | Дополнительный инструмент, доступный в расширенном режиме |

## Возможности

- Реализует инструмент 'think' согласно исследованиям Anthropic
- Минималистичный, основанный на стандартах MCP сервер использующий mcp[cli]
- Готов для интеграции с Claude или другими агентскими LLM
- Анализ результатов инструментов (обработка результатов предыдущих вызовов инструментов)
- Среды с множественными политиками (проверка соответствия рекомендациям)
- Последовательное принятие решений (где каждый шаг строится на предыдущих)
- Расширенный режим с дополнительными инструментами для рассуждений

## Переменные окружения

### Опциональные
- `TAVILY_API_KEY` - API ключ для функциональности поиска Tavily в расширенном режиме
