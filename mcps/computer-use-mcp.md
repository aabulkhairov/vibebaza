MCP сервер, который предоставляет Claude полный контроль над вашим компьютером, позволяя делать снимки экрана, кликать мышью и вводить текст с клавиатуры для автоматизации работы с рабочим столом и выполнения задач.

## Установка

### Ручная установка .dxt

```bash
1. Download computer-use-mcp-dxt from GitHub Actions
2. Rename .zip to .dxt
3. Double-click .dxt file
4. Click Install in Claude Desktop
```

### NPX

```bash
npx -y computer-use-mcp
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "computer-use": {
      "command": "npx",
      "args": [
        "-y",
        "computer-use-mcp"
      ]
    }
  }
}
```

### Cursor

```json
{
  "mcpServers": {
    "computer-use": {
      "command": "npx",
      "args": ["-y", "computer-use-mcp"]
    }
  }
}
```

### Cline

```json
{
  "mcpServers": {
    "computer-use": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "computer-use-mcp"]
    }
  }
}
```

## Возможности

- Полный контроль компьютера, аналогичный компьютерному использованию от Anthropic
- Захват экрана и визуальный анализ
- Автоматизация кликов мыши и движений курсора
- Ввод с клавиатуры и выполнение горячих клавиш
- Интеграция с расширением браузера Rango для надёжной навигации по сайтам
- Поддержка дисплеев высокого разрешения с рекомендациями по масштабированию

## Примечания

⚠️ ВНИМАНИЕ: Модели часто совершают ошибки и уязвимы для prompt-инъекций. Это даёт полный контроль над компьютером, поэтому используйте с осторожностью и рассмотрите возможность запуска в изолированной учётной записи пользователя. Лучшие результаты достигаются с установленным расширением браузера Rango для навигации с клавиатуры. Использует nut.js для взаимодействия с компьютером.
