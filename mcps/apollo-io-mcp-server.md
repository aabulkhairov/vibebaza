Реализация сервера Model Context Protocol (MCP) для бесшовной интеграции с Apollo.io API, позволяющая AI-ассистентам взаимодействовать с данными Apollo.io для обогащения информации о людях и организациях.

## Установка

### NPM

```bash
npm install @agentx-ai/apollo-io-mcp-server
```

### Тестирование локально

```bash
npm install -g .
npx @modelcontextprotocol/inspector apollo-io-mcp-server
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `people_enrichment` | Обогащение контактных данных всесторонней информацией, используя имя, фамилию, email, домен или... |
| `organization_enrichment` | Получение детальной информации о компании и аналитических данных по домену компании |
| `people_search` | Поиск людей по различным критериям, включая должности, локации, уровни старшинства и комп... |
| `organization_search` | Поиск компаний по определенным критериям с всесторонними опциями фильтрации, включая лока... |
| `organization_job_postings` | Доступ к вакансиям конкретных организаций по Apollo.io ID организации |
| `get_person_email` | Получение email-адреса человека по его Apollo ID |
| `employees_of_company` | Поиск сотрудников компании по названию компании, URL сайта или LinkedIn URL |

## Возможности

- Обогащение данных о людях: обогащение контактных данных всесторонней информацией
- Обогащение организационных данных: получение детальной информации о компании и аналитических данных
- Поиск людей: поиск людей по различным критериям
- Поиск организаций: поиск компаний по определенным критериям
- Вакансии: доступ к вакансиям конкретных организаций
- Поиск email: получение email-адресов по Apollo.io ID людей
- Сотрудники компании: поиск сотрудников конкретных компаний

## Переменные окружения

### Обязательные
- `APOLLO_IO_API_KEY` - API ключ для аутентификации Apollo.io
