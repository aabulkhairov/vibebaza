Универсальный инструмент для управления цитированием и академическими ссылками, который генерирует цитаты из DOI, arXiv, заголовков или URL в различных форматах (BibTeX, APA, MLA) с интеллектуальным дополнением метаданных.

## Установка

### PyPI

```bash
pip install onecite
```

### Тест MCP сервера

```bash
onecite-mcp
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": { "onecite": { "command": "onecite-mcp" } }
}
```

### Cursor

```json
{
  "mcpServers": { "onecite": { "command": "onecite-mcp" } }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `cite` | Генерирует одну цитату из DOI, arXiv ID, заголовка или URL |
| `batch_cite` | Обрабатывает список ссылок в пакетном режиме для множественных цитат |
| `search` | Ищет академическую литературу по ключевым словам в различных базах данных |

## Возможности

- Умное распознавание с нечетким поиском по множественным академическим базам данных
- Поддержка универсальных форматов - ввод txt/bib файлов, вывод в BibTeX, APA или MLA
- Высокоточный 4-этапный конвейер обработки (очистка → запрос → валидация → форматирование)
- Автоматическое дополнение недостающих метаданных как журнал, том, страницы, авторы
- Поддержка 7+ типов цитирования: журнальные статьи, конференционные доклады, книги, программное обеспечение, датасеты, диссертации, препринты
- Интеллектуальная маршрутизация с автоматическим определением типа контента и области
- Поддержка универсальных идентификаторов: DOI, PMID, arXiv ID, ISBN, GitHub URL, Zenodo DOI, обычный текст
- Интерактивный режим для ручного выбора при обнаружении множественных совпадений
- Пользовательские YAML-шаблоны для контроля формата вывода
- Интеграция с 9+ источниками данных: CrossRef, Semantic Scholar, OpenAlex, PubMed, dblp, arXiv, DataCite, Zenodo, Google Books

## Примечания

OneCite обрабатывает ссылки смешанных форматов из DOI, заголовков статей, arXiv ID, GitHub URL и запросов в виде обычного текста. Поддерживает интерактивное уточнение при обнаружении множественных совпадений и может использоваться как инструмент командной строки, так и библиотека Python.
