WildFly MCP сервер, который интегрируется с AI чат-ботами для взаимодействия с WildFly серверами на естественном языке. Позволяет выполнять задачи мониторинга и управления, включая получение метрик, логов и вызов операций.

## Возможности

- Взаимодействие с WildFly серверами на естественном языке
- Получение метрик с WildFly серверов
- Доступ к логам сервера
- Вызов операций управления
- Интерфейс чат-бота для управления WildFly
- Образы контейнеров для облачного деплоя
- Поддержка развертывания в OpenShift
- Шлюз протокола MCP STDIO to SSE
- Функционал ожидания для управления рабочими процессами

## Ресурсы

- [GitHub Repository](https://github.com/wildfly-extras/wildfly-mcp)

## Примечания

Это многокомпонентный проект, который включает: WildFly MCP сервер для AI интеграции, WildFly чат-бот с поддержкой MCP, образы контейнеров для облачного развертывания, шлюз MCP протокола и вспомогательные MCP серверы. Подробная документация доступна в файлах README отдельных компонентов.
