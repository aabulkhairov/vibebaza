Находите, понимайте и работайте с данными в watsonx.data intelligence — системе управления данными и каталогизации, контроля качества данных, отслеживания родословной данных и хабе продуктов данных

## Установка

### Pip

```bash
pip install ibm-watsonx-data-intelligence-mcp-server
```

## Переменные окружения

### Опциональные
- `DI_SERVICE_URL` - `None`
- `DI_ENV_MODE` - `SaaS`
- `REQUEST_TIMEOUT_S` - `60`
- `LOG_FILE_PATH` - `None`
- `DI_CONTEXT` - `df`
