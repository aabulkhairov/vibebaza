MCP сервер, который предоставляет точное время UTC и услуги конвертации часовых поясов с низкой задержкой и высокой надежностью, позволяя AI приложениям эффективно получать данные о текущем времени.

## Возможности

- Получение текущего времени UTC с точностью до миллисекунды
- Конвертация времени UTC в любой часовой пояс с использованием идентификаторов IANA
- Низкая задержка, оптимизированная для быстрого времени отклика
- Массовая конвертация - преобразование нескольких временных меток в одном запросе
- Стандартизированный интерфейс MCP для бесшовной интеграции
- Список доступных часовых поясов

## Ресурсы

- [GitHub Repository](https://github.com/jairampatel/currenttimeutc-mcp)

## Примечания

Это хостинговый сервис, доступный через REST API по адресу https://a.currenttimeutc.com/mcp. Репозиторий содержит только схему MCP и документацию, не код реализации бэкенда. Сервис поддерживает конвертацию часовых поясов с использованием идентификаторов IANA.
