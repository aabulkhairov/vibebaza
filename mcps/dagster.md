Экспериментальный API для создания проектов и компонентов Dagster, предоставляющий CLI инструменты для построения пайплайнов данных.

## Возможности

- Создание проектов Dagster
- Генерация компонентов
- CLI инструмент (`dg`) для управления проектами

## Ресурсы

- [GitHub Repository](https://github.com/dagster-io/dagster)

## Примечания

Это экспериментальный API, который включает CLI инструмент `dg` для создания проектов Dagster.
