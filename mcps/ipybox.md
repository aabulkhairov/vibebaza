Легкая и безопасная песочница для выполнения Python кода на основе IPython и Docker, предназначенная для AI агентов, которым нужно безопасно выполнять Python код для анализа данных или операций с кодом.

## Установка

### Python пакет

```bash
pip install ipybox
```

## Конфигурация

### Конфигурация MCP сервера

```json
{
  "mcpServers": {
    "ipybox": {
      "command": "uvx",
      "args": ["ipybox", "mcp"]
    }
  }
}
```

## Возможности

- Безопасное выполнение кода внутри Docker контейнеров
- Ограничение сетевого доступа с настраиваемым файрволом
- Статeful выполнение кода с IPython ядрами
- Потоковый вывод результатов выполнения кода в режиме реального времени
- Установка Python пакетов во время сборки или выполнения
- Возврат графиков, созданных с помощью библиотек визуализации
- Предоставляет MCP серверный интерфейс для интеграции с AI агентами
- Вызов MCP серверов через сгенерированный клиентский код
- Гибкие варианты деплоя, локально или удаленно
- asyncio API для управления средой выполнения
