Легкий сервер Model Context Protocol (MCP), который позволяет AI-ассистентам взаимодействовать с экспериментами, моделями и проектами ClearML для комплексного анализа контекста ML экспериментов.

## Установка

### PyPI

```bash
pip install clearml-mcp
```

### uvx (без установки)

```bash
uvx clearml-mcp
```

### Из исходников

```bash
git clone https://github.com/prassanna-ravishankar/clearml-mcp.git
cd clearml-mcp
uv sync
uv run python -m clearml_mcp.clearml_mcp
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_task_info` | Получить подробную информацию о задаче, параметры и статус |
| `list_tasks` | Список задач с расширенной фильтрацией (проект, статус, теги, пользователь) |
| `get_task_parameters` | Получить гиперпараметры и конфигурацию |
| `get_task_metrics` | Доступ к метрикам тренировки, скалярам и графикам |
| `get_task_artifacts` | Получить артефакты, файлы моделей и выходные данные |
| `get_model_info` | Получить метаданные модели и детали конфигурации |
| `list_models` | Просмотр доступных моделей с фильтрацией |
| `get_model_artifacts` | Доступ к файлам моделей и URL для загрузки |
| `list_projects` | Обнаружить доступные проекты ClearML |
| `get_project_stats` | Получить статистику проекта и сводку задач |
| `find_project_by_pattern` | Найти проекты, соответствующие шаблону имени |
| `find_experiment_in_project` | Найти конкретные эксперименты в проектах |
| `compare_tasks` | Сравнить несколько задач по определенным метрикам |
| `search_tasks` | Расширенный поиск по имени, тегам, комментариям и многому другому |

## Возможности

- Обнаружение экспериментов: Поиск и анализ ML экспериментов в проектах
- Анализ производительности: Сравнение метрик моделей и прогресса тренировки
- Метрики в реальном времени: Доступ к скалярам тренировки, кривым валидации и анализу сходимости
- Умный поиск: Фильтрация задач по имени, тегам, статусу и пользовательским запросам
- Управление артефактами: Получение файлов моделей, датасетов и результатов экспериментов
- Кроссплатформенность: Работает со всеми основными AI-ассистентами и редакторами кода

## Примечания

Требует аккаунт ClearML с валидными API credentials в ~/.clearml/clearml.conf. Предоставляет 14 комплексных инструментов для анализа ML экспериментов. Совместим с Zed Editor, OpenHands, Roo-Cline и любыми приложениями с поддержкой MCP.
