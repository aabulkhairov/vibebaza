Сервис SEO инструментов Model Context Protocol, который предоставляет комплексный анализ SEO данных, включая анализ обратных ссылок, исследование ключевых слов, оценку трафика и расчет сложности ключевых слов с использованием данных Ahrefs.

## Установка

### PyPI

```bash
pip install seo-mcp
```

### UV

```bash
uv pip install seo-mcp
```

### Из исходного кода

```bash
git clone https://github.com/cnych/seo-mcp.git
cd seo-mcp
pip install -e .
# Или
uv pip install -e .
```

```json
{
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_backlinks_list` | Получение детальных данных обратных ссылок для любого домена, включая рейтинг домена, анкорный текст и атрибуты ссылок |
| `keyword_generator` | Генерация идей ключевых слов из базового ключевого слова с оценками сложности и объемами поиска |
| `get_traffic` | Получение оценки трафика, истории, трендов, популярных страниц и анализа распределения по странам |
| `keyword_difficulty` | Получение оценки сложности ключевого слова с анализом SERP и связанными ключевыми словами |

## Возможности

- Детальный анализ обратных ссылок с рейтингом домена и анкорным текстом
- Фильтрация образовательных и правительственных доменов в данных обратных ссылок
- Генерация идей ключевых слов с оценками сложности и объемами поиска
- Оценка трафика веб-сайта и анализ трендов
- Отслеживание популярных страниц и распределения по странам
- Мониторинг позиций ключевых слов
- Автоматическое решение CAPTCHA с интеграцией CapSolver
- Кэширование ответов для улучшения производительности

## Примечания

Требует Python 3.10 или выше и аккаунт CapSolver. Этот сервис предназначен только для образовательных целей и использует данные Ahrefs через автоматизированные процессы, включая решение CAPTCHA и аутентификацию.
