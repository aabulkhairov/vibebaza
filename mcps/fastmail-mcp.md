MCP сервер, который предоставляет доступ к Fastmail API, позволяя ИИ-ассистентам взаимодействовать с электронной почтой, контактами и календарем через протокол JMAP.

## Установка

### Из исходного кода

```bash
npm install
npm run build
npm start
```

### NPX (GitHub)

```bash
FASTMAIL_API_TOKEN="your_token" FASTMAIL_BASE_URL="https://api.fastmail.com" \
  npx --yes github:MadLlama25/fastmail-mcp fastmail-mcp
```

### NPX (релиз с тегом)

```bash
FASTMAIL_API_TOKEN="your_token" \
  npx --yes github:MadLlama25/fastmail-mcp@v1.6.1 fastmail-mcp
```

### Расширение Claude Desktop (DXT)

```bash
npm run build
npx dxt pack
# Затем откройте .dxt файл или перетащите его в Claude Desktop
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `check_function_availability` | Проверить доступные функции и получить руководство по настройке |
| `test_bulk_operations` | Безопасно протестировать массовые операции в режиме dry-run |
| `send_email` | Полнофункциональная отправка писем с корректной обработкой черновиков/отправленных |
| `advanced_search` | Мощная фильтрация писем по нескольким критериям |
| `get_recent_emails` | Быстрый доступ к последним письмам из любого почтового ящика |
| `list_mailboxes` | Получить все почтовые ящики в вашем аккаунте |
| `list_emails` | Список писем из конкретного почтового ящика или всех ящиков |
| `get_email` | Получить конкретное письмо по ID |
| `search_emails` | Поиск писем по содержимому |
| `mark_email_read` | Отметить письмо как прочитанное или непрочитанное |
| `delete_email` | Удалить письмо (переместить в корзину) |
| `move_email` | Переместить письмо в другой почтовый ящик |
| `get_email_attachments` | Получить список вложений письма |
| `download_attachment` | Получить ссылку для скачивания вложения письма |
| `get_thread` | Получить все письма в цепочке разговора |

## Возможности

- Просмотр почтовых ящиков и получение их статистики
- Просмотр, поиск и фильтрация писем по расширенным критериям
- Получение конкретных писем по ID с полным содержимым
- Отправка писем (текст и HTML) с корректной обработкой черновиков/отправленных
- Управление письмами: отметка прочитанными/непрочитанными, удаление, перемещение между папками
- Работа с вложениями: список и скачивание вложений писем
- Поддержка цепочек: получение полных цепочек разговоров
- Расширенный поиск: фильтрация по нескольким критериям (отправитель, диапазон дат, вложения, статус прочтения)
- Массовые операции: обработка нескольких писем одновременно
- Статистика и аналитика: сводки аккаунта и статистика почтовых ящиков

## Переменные окружения

### Обязательные
- `FASTMAIL_API_TOKEN` - Ваш токен Fastmail API для аутентификации

### Опциональные
- `FASTMAIL_BASE_URL` - Настройка базового URL (по умолчанию https://api.fastmail.com)

## Примечания

Этот сервер использует протокол JMAP от Fastmail. Функции вдохновлены репозиторием Fastmail JMAP-Samples. Функциональность календаря и контактов может требовать бизнес/профессиональных планов Fastmail и соответствующих разрешений токена API. Сервер включает 31 инструмент с встроенными возможностями тестирования и dry-run для безопасных операций.
