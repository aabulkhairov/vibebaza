MCP сервер (Model Context Protocol), который предоставляет AI-ассистентам безопасный доступ к финансовым данным YNAB (You Need A Budget) для анализа, запросов по бюджету, анализа трат и отслеживания финансовых целей.

## Установка

### Smithery

```bash
npx -y @smithery/cli install @ChuckBryan/ynabmcpserver --client claude
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `GetUserInfo` | Получить ваш YNAB пользовательский ID |
| `ListBudgets` | Получить все бюджеты в вашем YNAB аккаунте |
| `GetBudgetDetails` | Подробная информация о конкретном бюджете |
| `GetBudgetMonths` | Список месяцев бюджета |
| `ListCategories` | Все категории в бюджете |
| `GetCategoryDetails` | Подробная информация о категории |
| `SearchTransactions` | Найти транзакции с настраиваемыми фильтрами |
| `ListAccountTransactions` | Получить транзакции для конкретного счета |
| `ListAccounts` | Получить все счета в бюджете |
| `ListPayees` | Получить всех получателей платежей в бюджете |
| `GetCurrentMonthSnapshot` | Сводка бюджета текущего месяца |
| `GetRecentActivitySummary` | Сводка недавней финансовой активности |
| `GetIncomeVsExpenseSummary` | Сравнить доходы и расходы за период |

## Возможности

- Запрос информации о вашем бюджете
- Анализ паттернов трат
- Проверка балансов счетов
- Отслеживание финансовых целей
- Получение инсайтов о ваших финансовых привычках
- Комплексный набор инструментов для взаимодействия с YNAB API
- Получение информации о пользователе и бюджете
- Управление категориями и транзакциями
- Доступ к данным счетов и получателей платежей
- Финансовый анализ и отчетность

## Примечания

Требует аккаунт YNAB с доступом к API и MCP-совместимый клиент. Сервер работает локально и обеспечивает безопасность вашего YNAB API токена, не делясь им с третьими сторонами. Docker образ доступен для платформ linux/amd64 и linux/arm64. Включает интеграцию с VS Code с кнопками установки в один клик.
