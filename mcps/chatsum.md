MCP сервер, который запрашивает и суммирует сообщения чата из локальной базы данных чата, позволяя пользователям анализировать историю своих разговоров.

## Установка

### Из исходного кода

```bash
pnpm install
pnpm build
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "mcp-server-chatsum": {
      "command": "path-to/bin/node",
      "args": ["path-to/mcp-server-chatsum/build/index.js"],
      "env": {
        "CHAT_DB_PATH": "path-to/mcp-server-chatsum/chatbot/data/chat.db"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `query_chat_messages` | Запрос сообщений чата с заданными параметрами и суммирование сообщений чата на основе запроса |

## Возможности

- Запрос сообщений чата из локальной базы данных
- Суммирование сообщений чата на основе запросов
- Интеграция с системой чат-бота для сохранения сообщений чата

## Переменные окружения

### Обязательные
- `CHAT_DB_PATH` - Путь к файлу базы данных чата

## Ресурсы

- [GitHub Repository](https://github.com/mcpso/mcp-server-chatsum)

## Примечания

Требуется предварительная настройка директории чат-бота и запуск чат-бота для сохранения сообщений чата. Доступен режим разработки с 'pnpm watch' для автоматической пересборки. Отладка поддерживается через MCP Inspector с помощью команды 'pnpm inspector'.
