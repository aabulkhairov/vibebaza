MCP сервер для доступа к публичному API польского Национального судебного реестра (KRS), позволяющий запрашивать текущую и историческую информацию о бизнес-сущностях.

## Установка

### NPX

```bash
npx krs-poland-mcp-server
```

### Из исходников

```bash
git clone repository
npm install
npm run build
npm start
```

## Конфигурация

### LLM клиенты (LM Studio)

```json
{
  "mcpServers": {
    "krs-poland-mcp-server": {
      "command": "npx",
      "args": [
        "-y",
        "krs-poland-mcp-server"
      ]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get-krs-current-extract` | Получает текущий статус и информацию для заданной сущности из KRS |
| `get-krs-full-extract` | Получает полные исторические данные для заданной сущности из KRS |

## Возможности

- Доступ к публичному API польского Национального судебного реестра
- Получение текущего статуса бизнес-сущностей
- Доступ к полным историческим данным для сущностей
- Не требует глобальной установки при использовании npx
