Кураторская подборка высококачественных удаленных серверов Model Context Protocol (MCP), доступных через интернет, с фокусом на официальные и хорошо поддерживаемые сервисы для AI-приложений.

## Возможности

- Кураторская подборка 80+ удаленных MCP серверов в различных категориях
- Поддержка аутентификации включая OAuth 2.1, API Key и открытый доступ
- Покрывает категории как управление проектами, CRM, разработка ПО, платежи, аналитика
- Готовые к продакшену серверы с активной поддержкой и официальным сопровождением
- Фокус на безопасности с правильными мерами аутентификации
- Совместимость с веб-базируемыми MCP клиентами
- Простое копирование-вставка URL серверов для быстрой настройки

## Примечания

Это директория/список удаленных MCP серверов, а не единственная реализация MCP сервера. Каждый указанный сервер имеет свой собственный URL и требования аутентификации. Удаленные серверы предлагают лучшую безопасность и простоту использования по сравнению с локальными установками, и являются единственным способом использования MCP с веб-базируемыми клиентами. Репозиторий поддерживает высокие стандарты качества, требуя официальную поддержку, готовность к продакшену, активное сопровождение, правильную безопасность и проверенную надежность.
