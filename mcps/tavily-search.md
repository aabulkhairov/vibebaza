MCP сервер, который предоставляет возможности веб-поиска с поддержкой ИИ через API Tavily, позволяя LLM выполнять сложные веб-поиски, получать прямые ответы и искать последние новостные статьи с извлечённым ИИ контентом.

## Установка

### pip

```bash
pip install mcp-tavily
```

### uv

```bash
uv add mcp-tavily
```

### Из исходников

```bash
git clone https://github.com/RamXX/mcp-tavily.git
cd mcp-tavily
python -m venv .venv
source .venv/bin/activate
uv sync
uv build
```

### Docker

```bash
docker build -t mcp_tavily .
docker run -d --name mcp_tavily_container -e TAVILY_API_KEY=your_api_key_here -p 8000:8000 mcp_tavily
```

## Конфигурация

### Claude Desktop

```json
"mcpServers": {
  "tavily": {
    "command": "python",
    "args": ["-m", "mcp_server_tavily"]
  },
  "env": {
    "TAVILY_API_KEY": "your_api_key_here"
  }
}
```

### VS Code MCP

```json
{
  "mcp": {
    "inputs": [
      {
        "type": "promptString",
        "id": "apiKey",
        "description": "Tavily API Key",
        "password": true
      }
    ],
    "servers": {
      "tavily": {
        "command": "uvx",
        "args": ["mcp-tavily"],
        "env": {
          "TAVILY_API_KEY": "${input:apiKey}"
        }
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `tavily_web_search` | Выполняет комплексные веб-поиски с извлечением контента на основе ИИ |
| `tavily_answer_search` | Выполняет веб-поиски и генерирует прямые ответы с подтверждающими доказательствами |
| `tavily_news_search` | Ищет последние новостные статьи с датами публикации |

## Возможности

- Веб-поиск с поддержкой ИИ и извлечением контента
- Генерация прямых ответов с подтверждающими доказательствами
- Поиск последних новостных статей с датами публикации
- Фильтрация по включению и исключению доменов
- Настраиваемая глубина поиска (базовая или продвинутая)
- Шаблоны запросов для каждого типа поиска
- Поддержка до 20 результатов на поиск

## Переменные окружения

### Обязательные
- `TAVILY_API_KEY` - API ключ для сервиса поиска Tavily

## Примечания

Этот сервер устарел в пользу официального MCP сервера от Tavily. Сервер поддерживает комплексное тестирование с автоматизированным тестированием совместимости зависимостей и включает поддержку Docker с make командами для простого деплоя.
