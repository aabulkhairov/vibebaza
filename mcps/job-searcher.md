Реализация MCP сервера, который предоставляет функциональность поиска работы с возможностями фильтрации по временному периоду, ключевым словам и предпочтениям удаленной работы.

## Установка

### NPX

```bash
npx -y job-searchoor
```

## Конфигурация

### Claude Desktop

```json
{
"mcpServers": {
    "job-search": {
        "command": "npx",
        "args": ["-y", "job-searchoor"]
    }
}
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_jobs` | Получить доступные вакансии с опциями фильтрации включая временной период, ключевые слова для включения/исключения и... |

## Возможности

- Фильтрация вакансий по временному периоду (дни и недели)
- Поиск вакансий по ключевым словам
- Исключение вакансий по определенным ключевым словам
- Фильтрация вакансий по доступности удаленной работы

Лицензировано под лицензией MIT. Фильтрация по времени поддерживает формат дней и недель (например, '1d' или '1w').
