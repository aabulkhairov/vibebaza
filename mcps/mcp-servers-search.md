MCP сервер, который предоставляет инструменты для поиска и обнаружения доступных MCP серверов из официального репозитория modelcontextprotocol/servers с возможностями поиска, фильтрации и открытия более 547 MCP серверов.

## Установка

### NPM Global

```bash
npm install -g @atonomus/mcp-servers-search
```

### Из исходного кода

```bash
git clone https://github.com/atonomus/mcp-servers-search.git
cd mcp-servers-search
npm install
```

### NPX напрямую

```bash
npx @atonomus/mcp-servers-search
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "servers-search": {
      "command": "npx",
      "args": ["-y", "@atonomus/mcp-servers-search"]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `list_servers` | Показать все доступные MCP серверы с опциональной фильтрацией по категории, поисковым терминам и лимиту |
| `get_server_details` | Получить подробную информацию о конкретном MCP сервере по имени |
| `search_servers_by_feature` | Искать серверы, которые предоставляют определенные функции или возможности |
| `get_random_servers` | Получить случайную выборку MCP серверов для исследования |
| `refresh_server_list` | Принудительно обновить кешированный список MCP серверов с GitHub |

## Возможности

- Просмотр всех доступных MCP серверов с фильтрацией по категориям
- Поиск серверов по имени, описанию или автору
- Обнаружение серверов, которые предоставляют определенные возможности
- Получение случайных предложений серверов для исследования
- Эффективное кеширование для минимизации вызовов GitHub API

## Примечания

Сервер кеширует список доступных MCP серверов на 1 час для минимизации вызовов API к GitHub. Данные получены из официального репозитория MCP серверов и включают комплексный набор тестов, покрывающий различные сценарии парсинга README.
