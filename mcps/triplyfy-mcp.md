Удаленный MCP сервер, который позволяет Claude планировать, редактировать и сохранять поездки в Triplyfy — приложение для планирования путешествий с созданием маршрутов и интерактивными картами.

## Конфигурация

### Claude Desktop

```json
Open Claude Desktop → Settings → Developer → MCP Servers → Add server
- Name: `Triplyfy`
- MCP SSE endpoint: `https://triplyfy.com/mcp/sse`
```

## Возможности

- Планирование, редактирование и сохранение поездок в Triplyfy
- Создание и публикация маршрутов с публичными ссылками
- Перетаскивание мест и заметок между днями
- Просмотр всего на интерактивной карте
- Добавление рейсов в маршруты
- Поиск рейсов
- Google OAuth аутентификация с токенами сессии на 30 дней

## Ресурсы

- [GitHub Repository](https://github.com/helpful-AIs/triplyfy-mcp)

## Примечания

Это удаленный MCP сервер, который требует входа через Google при первом подключении. Сервер использует OAuth для аутентификации и выполняет только действия, запрашиваемые пользователем. Ручная настройка шлюза не требуется.
