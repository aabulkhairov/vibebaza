MCP сервер, предоставляющий serverless облачную инфраструктуру для AI агентов через платформу Alibaba Cloud Wuying AgentBay. Браузерная автоматизация, файловые операции и терминальный доступ в безопасной облачной среде.

## Конфигурация

### Cursor SSE метод

```json
{
  "mcpServers": {
    "wuying_mcp_server": {
      "url": "https://agentbay.wuying.aliyun.com/sse?APIKEY=YOUR_API_KEY"
    }
  }
}
```

### Cursor SSE метод с External ID

```json
{
  "mcpServers": {
    "wuying_mcp_server": {
      "url": "https://agentbay.wuying.aliyun.com/sse?APIKEY=YOUR_API_KEY&EXTERNALID=user001"
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `Browser` | Автоматизация и управление браузером в облачной среде |
| `File` | Файловые операции и управление в облачной среде |
| `Terminal` | Командная строка и терминальные операции в облачной среде |

## Возможности

- Стандартная runtime среда с пре-интегрированными инструментами для выполнения задач агентов
- Персистентность состояния пользователя с безопасной изоляцией конфигураций и cookies
- Взаимодействие edge-cloud в реальном времени через проприетарный ASP протокол
- Serverless возможности с управлением сессиями окружения в один клик
- Масштабирование до 10,000 параллельных операций за секунды
- Поддержка стриминга экрана для визуального взаимодействия с облачным компьютером
- Enterprise-grade безопасность и инфраструктура
- Глобальное развёртывание с низкой задержкой доступа

## Переменные окружения

### Обязательные
- `APIKEY` — API ключ для аутентификации в сервисе Wuying AgentBay

### Опциональные
- `EXTERNALID` — опциональный external ID для сохранения конфигурации и окружения между сессиями

## Примеры использования

```
open the browser with wuying-agentbay and access wuying.aliyun.com
```

## Примечания

Требуется создать API Key через AgentBay Console на https://agentbay.console.aliyun.com/service-management. Сейчас в публичной бете с лимитом 10 API Keys и 10 параллельных инстансов. Стриминг экрана предоставляет веб-ссылки для визуального взаимодействия с облачным окружением. Поддерживаются оба метода установки: SSE и STDIO.
