Инструмент управления целями, который превращает традиционное отслеживание целей в увлекательный нарративный опыт с использованием разговорного ИИ и техник визуализации для помощи пользователям сосредоточиться на одной цели за раз.

## Установка

### NPX

```bash
npx -y goalstory-mcp
```

## Конфигурация

### Конфигурация MCP сервера

```json
"goalStory": {
    "command": "npx",
    "args": ["-y", "goalstory-mcp", "https://prod-goalstory-rqc2.encr.app", "your-api-key"]
}
```

## Возможности

- Фокус на одной цели за раз вместо жонглирования бесконечными списками
- Управление целями с помощью разговорного ИИ
- Креативное повествование, адаптированное под личную мотивацию
- Техники визуализации, интегрированные в процесс планирования
- Движок целей, который превращает отслеживание в увлекательный процесс
- Безопасное сохранение инсайтов и прогресса для будущей рефлексии
- Интеграция с существующими инструментами отслеживания задач

## Переменные окружения

### Обязательные
- `API_KEY` - Ключ аутентификации от GoalStory.ing для доступа к Goal Story API

## Ресурсы

- [GitHub Repository](https://github.com/hichana/goalstory-mcp)

## Примечания

Требует API ключ от GoalStory.ing (доступна бесплатная регистрация). Основан на исследованиях, показывающих, что мысленные образы усиливают намерения реализации для достижения более высоких показателей успеха в целях. Работает как дополнение к существующим инструментам отслеживания задач, а не их замена.
