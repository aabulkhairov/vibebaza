MCP сервер, который предоставляет данные индекса страха и жадности в криптовалютах в реальном времени и исторические данные, работающий на базе Alternative.me, что делает простой интеграцию анализа настроений криптовалютного рынка в совместимые с MCP клиенты.

## Установка

### Smithery

```bash
npx -y @smithery/cli install @kukapay/crypto-feargreed-mcp --client claude
```

### Из исходников

```bash
git clone https://github.com/kukapay/crypto-feargreed-mcp.git
cd crypto-feargreed-mcp
```

### Claude Desktop

```bash
mcp install main.py --name "CryptoFearGreed"
```

## Конфигурация

### Другие MCP клиенты

```json
"mcpServers": {
  "crypto-feargreed-mcp": {
    "command": "uv",
    "args": [
      "--directory", "/your/path/to/crypto-feargreed-mcp",
      "run",
      "main.py"
    ]
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_current_fng_tool` | Получить последнее значение индекса страха и жадности и его классификацию |
| `get_historical_fng_tool` | Получить исторические значения индекса за указанное количество дней |
| `analyze_fng_trend` | Анализировать тренды во времени со статистикой, такой как среднее значение и направление тренда |

## Возможности

- Текущий индекс: получение последнего значения индекса страха и жадности и его классификации
- Исторические данные: получение исторических значений индекса за указанное количество дней
- Анализ трендов: анализ трендов во времени со статистикой, такой как среднее значение и направление тренда
- Поддержка только инструментов: включает версии инструментов для всех ресурсов для совместимости с MCP клиентами, работающими только с инструментами
- Генерация промптов: предоставляет шаблон промпта для интерпретации значений индекса

## Примечания

Сервер предоставляет как ресурсы (fng://current, fng://history/{days}), так и инструменты для той же функциональности. Он включает шаблон промпта 'interpret_fng' для интерпретации данных индекса. Данные предоставляются API индекса страха и жадности Alternative.me.
