MCP сервер с интеграцией Financial Modeling Prep API для продвинутого финансового анализа, созданный специально для аналитиков инвестиционного банкинга и профессионалов M&A.

## Установка

### NPM Global

```bash
npm install -g @vipbat/fmp-mcp-server
```

### Python зависимости

```bash
pip install mcp fastmcp httpx
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "fmp": {
      "command": "npx",
      "args": ["@vipbat/fmp-mcp-server"],
      "env": {
        "FMP_API_KEY": "your-fmp-api-key-here"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_company_profile` | Получает исчерпывающую информацию о компании, включая сектор, отрасль, рыночную капитализацию и описание |
| `get_income_statements` | Извлекает отчеты о прибылях и убытках с показателями доходов, расходов и прибыли |
| `get_balance_sheets` | Получает данные баланса, включая активы, обязательства и капитал |
| `get_cash_flow_statements` | Предоставляет отчеты о движении денежных средств, показывающие операционную, инвестиционную и финансовую деятельность |
| `get_ratios` | Получает финансовые коэффициенты, включая P/E, P/B, ROE, ROA и другие |
| `get_enterprise_value` | Получает расчеты стоимости предприятия и связанные метрики |
| `search_companies_by_name` | Поиск компаний по названию для нахождения тикер-символов |

## Возможности

- Профили компаний - Получение детальной информации о компании, включая сектор, отрасль и описания
- Финансовые отчеты - Доступ к отчетам о прибылях и убытках, балансам и отчетам о движении денежных средств
- Финансовые коэффициенты - Извлечение ключевых метрик, таких как коэффициент P/E, ROE, ROA и других
- Стоимость предприятия - Получение расчетов стоимости предприятия и мультипликаторов
- Поиск компаний - Поиск компаний по названию для нахождения тикер-символов
- M&A анализ - Проведение сравнительного анализа компаний и оценки секторов

## Переменные окружения

### Обязательные
- `FMP_API_KEY` - API ключ Financial Modeling Prep для доступа к финансовым данным
