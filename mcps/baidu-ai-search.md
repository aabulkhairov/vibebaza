Компонент Baidu AI Search, который объединяет поисковые возможности Baidu с технологией больших языковых моделей для предоставления интеллектуальных ответов со ссылками на актуальную веб-информацию в реальном времени.

## Конфигурация

### Конфигурация MCP сервера

```json
{
  "mcpServers": {
    "baidu_ai_search": {
      "url": "http://appbuilder.baidu.com/v2/ai_search/mcp/sse?api_key=Bearer+bce-v3/ALTAK..."
    }
  }
}
```

## Возможности

- Настройка персонализированного поведения
- Выбор модели
- Перезапись запросов (включая временно-зависимые и многооборотные подходы для улучшения результатов поиска)
- Конфигурация области поиска (выбор модальностей, диапазона сайтов и дат публикации)
- Настраиваемое количество ссылок-источников
- Ссылки на веб-информацию в реальном времени
- Поддержка различных сценариев применения в индустрии

## Ресурсы

## Примечания

Требуется AppBuilder API Key из консоли Baidu. Токен авторизации должен быть отформатирован как 'Bearer+<AppBuilder API Key>' с обязательным включением символа '+'. Дополнительная документация доступна по адресу https://cloud.baidu.com/doc/AppBuilder/s/zm8pn5cju
