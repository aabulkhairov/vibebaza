MCP сервер, который реализует инструмент 'think' для улучшения способностей сложного рассуждения у больших языковых моделей, предоставляя LLM выделенное пространство для структурированного мышления при решении задач.

## Установка

### NPX (Рекомендуется)

```bash
npx -y think-tool-mcp
```

### Глобальная установка

```bash
npm install -g think-tool-mcp
```

### Из исходного кода

```bash
git clone https://github.com/abhinav-mangla/think-tool-mcp.git
cd think-tool-mcp
npm install
npm run build
npm start
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "think-tool": {
      "command": "npx",
      "args": ["-y", "think-tool-mcp"]
    }
  }
}
```

## Возможности

- Структурированное пространство для мышления: Предоставляет LLM выделенную среду для сложного рассуждения
- Помощь с памятью: Помогает поддерживать контекст во время длинных цепочек вызовов инструментов
- Проверка политик: Позволяет тщательно проверять соблюдение политик
- Декомпозиция проблем: Поддерживает разбиение сложных проблем на шаги
- Легковесность: Минимальные накладные расходы с эффективной MCP реализацией
- Простая интеграция: Легкая настройка с популярными AI платформами (Cursor, Claude Desktop и др.)
- TypeScript: Построен с TypeScript для безопасности типов и лучшего опыта разработки
- Универсальная совместимость: Работает с любой LLM, которая поддерживает Model Context Protocol

## Примечания

Основан на исследовании Anthropic, показывающем улучшение производительности до 54% в задачах сложного рассуждения. Требует Node.js 16+ и работает с Cursor IDE (версия 0.45.6+), Claude Desktop и другими MCP-совместимыми платформами. Улучшения производительности продемонстрированы на бенчмарках τ-Bench и SWE-Bench.
