Практический менеджер списка задач с интеграцией MCP, который позволяет AI-ассистентам управлять задачами через естественные языковые взаимодействия. Обеспечивает локальное хранение без необходимости в внешних SaaS аккаунтах.

## Установка

### NPX

```bash
npx -y todos-mcp
```

### Из исходного кода

```bash
git clone https://github.com/tomelliot/todos-mcp.git
cd mcp-todo-app
npm install
# или
yarn install

npm run dev
# или
yarn dev
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    // ...
    "todos": {
      "command": "npx",
      "args": ["-y", "todos-mcp"]
    }
    // ...
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `Get-Todo` | Получить конкретную задачу по ID |
| `List-All-Todos` | Список всех задач |
| `List-Completed-Todos` | Список всех выполненных задач |
| `List-Pending-Todos` | Список всех ожидающих выполнения задач |
| `Add-Todo` | Добавить задачу |
| `Mark-Todo-Done` | Отметить задачу как выполненную |
| `List-Due-Today` | Список всех ожидающих задач, которые должны быть выполнены сегодня или просрочены |
| `List-Due-This-Week` | Список всех ожидающих задач, которые должны быть выполнены на этой неделе или раньше |
| `Update-Todo` | Обновить описание и/или срок выполнения задачи |

## Возможности

- Полная функциональность списка задач (создание, чтение, обновление, удаление задач)
- MCP-совместимый API для интеграции с AI-ассистентами
- Локальное хранение (не требует SaaS аккаунта)

## Примечания

Создан с использованием современных веб-технологий и реализует спецификацию MCP для интеграции с AI. Требует Node.js версии 16 или выше. Расположение файла конфигурации для Claude Desktop на MacOS: ~/Library/Application Support/Claude/claude_desktop_config.json
