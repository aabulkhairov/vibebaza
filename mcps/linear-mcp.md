MCP сервер для Linear, который предоставляет AI-ассистентам полный доступ к возможностям управления проектами Linear, включая задачи, проекты, команды, пользователей, дорожные карты, документы и инициативы.

## Установка

### Из исходного кода

```bash
git clone https://github.com/yourusername/linear-mcp.git
cd linear-mcp
npm install
npm run build
```

### Режим разработки

```bash
npm run dev
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "linear": {
      "command": "node",
      "args": [
        "/path/to/linear-mcp/dist/index.js"
      ],
      "env": {
        "LINEAR_API_KEY": "your_linear_api_key_here"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `create_issue` | Создание новых задач в Linear |
| `update_issue` | Обновление существующих задач |
| `add_comment` | Добавление комментариев к задачам |
| `create_project` | Создание новых проектов |
| `update_project` | Обновление существующих проектов |
| `add_issues_to_project` | Добавление задач в проекты |
| `create_initiative` | Создание инициатив |
| `link_project_to_initiative` | Связывание проектов с инициативами |
| `unlink_project_from_initiative` | Отвязка проектов от инициатив |
| `search` | Комплексный поиск по всем объектам Linear |

## Возможности

- Доступ к задачам, проектам, командам, пользователям, дорожным картам, документам, инициативам и другим ресурсам
- Создание и обновление задач, управление проектами, поиск, связывание проектов с инициативами
- Шаблоны для создания задач, отчетов об ошибках, запросов новых функций
- Поддержка stdio транспорта и HTTP сервера с SSE
- Ресурсы для конкретных команд, включая задачи, статусы, метки, циклы и участников
- Ресурсы для конкретных проектов и управление инициативами
- Доступ на уровне организации, включая сервисы подписки и аутентификации

## Переменные окружения

### Обязательные
- `LINEAR_API_KEY` - Ваш API ключ Linear

### Опциональные
- `SERVER_PORT` - Порт для HTTP сервера
- `LOG_LEVEL` - Уровень логирования

## Примечания

Поддерживает как stdio транспорт для CLI инструментов, так и режим HTTP сервера для удаленных подключений. Можно тестировать с помощью MCP Inspector. Требует Node.js 18 или новее и API ключ Linear.
