Корпоративная MCP платформа, которая обеспечивает централизованную оркестрацию, безопасность, наблюдаемость и управление затратами для MCP серверов в организациях.

## Установка

### Docker

```bash
docker pull archestra/platform:latest;
docker run -p 9000:9000 -p 3000:3000 \
  -v archestra-postgres-data:/var/lib/postgresql/data \
  -v archestra-app-data:/app/data \
  archestra/platform;
```

## Возможности

- Централизованная оркестрация MCP серверов на Kubernetes
- Приватный MCP реестр с управлением для команд
- ChatGPT-подобный интерфейс с общекорпоративным реестром промптов
- Агенты безопасности для предотвращения prompt injection атак
- Непероятностная защита от утечки данных
- Мониторинг затрат, лимиты и динамическая оптимизация (до 96% снижения затрат)
- Наблюдаемость AI с метриками, трейсами и логами
- Отслеживание использования по командам, агентам и организациям
- Молниеносная производительность (45ms на 95-м перцентиле)
- Terraform провайдер для инфраструктуры как код

## Примечания

Платформа включает Web UI на localhost:3000 и прокси на localhost:9000. Поддерживает интеграции с LangChain, Mastra.ai, N8N, OpenWebUI, Pydantic AI и Vercel AI. Является частью CNCF Linux Foundation. Готова для продакшена с корпоративными функциями безопасности.
