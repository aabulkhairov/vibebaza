Реализация MCP сервера, который интегрирует Hyperliquid SDK для предоставления доступа к данным криптовалютной биржи, включая средние цены, свечные данные и информацию из книги ордеров.

## Установка

### NPX

```bash
npx -y @mektigboy/server-hyperliquid
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "brave-search": {
      "command": "npx",
      "args": ["-y", "@mektigboy/server-hyperliquid"]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_all_mids` | Получение средних цен для всех монет на Hyperliquid |
| `get_candle_snapshot` | Получение исторических свечных данных для любого токена с настраиваемыми временными интервалами и диапазонами дат |
| `get_l2_book` | Доступ к книге ордеров L2 для любого токена |

## Возможности

- Получение средних цен для всех монет на Hyperliquid
- Получение исторических свечных данных с настраиваемыми временными интервалами
- Доступ к данным книги ордеров L2 для любого токена
- Интеграция с API биржи Hyperliquid
