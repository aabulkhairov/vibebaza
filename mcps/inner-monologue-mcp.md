Инструмент когнитивного мышления, который позволяет большим языковым моделям заниматься приватной структурированной саморефлексией и многоэтапными рассуждениями перед генерацией ответов, улучшая качество ответов и способности к решению проблем.

## Установка

### NPM Global Install

```bash
npm install -g inner-monologue-mcp
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "inner-monologue": {
      "command": "npx",
      "args": ["inner-monologue-mcp"]
    }
  }
}
```

### Cursor

```json
{
  "name": "inner-monologue",
  "command": "npx",
  "args": ["inner-monologue-mcp"]
}
```

### VS Code

```json
{
  "mcp.servers": {
    "inner-monologue": {
      "command": "npx",
      "args": ["inner-monologue-mcp"],
      "env": {}
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `inner-monologue` | Инструмент для внутренних рассуждений и рефлексии, который обрабатывает мысли без создания видимого ре... |

## Возможности

- Скрытая обработка: Мысли обрабатываются внутренне без влияния на внешний вывод
- Структурированные рассуждения: Поддерживает сложные многоэтапные процессы рассуждений
- Гибкий ввод: Принимает любые формы внутренних рассуждений, от отладки до решения математических задач
- MCP интеграция: Плавно интегрируется с Claude и другими MCP-совместимыми клиентами
- Улучшенное разложение проблем: Разбиение сложных проблем на управляемые шаги
- Предотвращение ошибок: Обнаружение логических несоответствий до генерации ответов
- Верификация решений: Мысленное тестирование подходов перед реализацией
- Сохранение контекста: Поддержание цепочек рассуждений через многоэтапные проблемы

## Примечания

Вдохновлен исследовательской работой Inner Monologue от Google DeepMind. Отличается от Think Tool использованием внутреннего диалога от первого лица вместо аналитической документации от третьего лица. Требует Node.js v18.x или выше. Улучшения производительности включают ~50% ускорение отладки кода, +25% точности в математических задачах и +40% полноты в сложных задачах планирования.
