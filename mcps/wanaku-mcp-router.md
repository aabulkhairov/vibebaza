Wanaku — это MCP роутер, который обеспечивает централизованную маршрутизацию и управление ресурсами для AI агентов, выступая в роли шлюза между несколькими MCP серверами с обширными возможностями интеграции.

## Установка

### JBang

```bash
jbang app install wanaku@wanaku-ai/wanaku
```

### Загрузка бинарного файла

```bash
# Download the latest binary from releases
# https://github.com/wanaku-ai/wanaku/releases
```

## Возможности

- Унифицированный доступ - Централизованная маршрутизация и управление ресурсами для AI агентов
- MCP-to-MCP мост - Действует как шлюз или прокси для других MCP серверов
- Обширная связность - Использует 300+ компонентов Apache Camel для интеграции
- Безопасность по умолчанию - Встроенная аутентификация и авторизация через Keycloak
- Kubernetes-совместимость - Первоклассная поддержка развертывания в OpenShift и Kubernetes
- Расширяемая архитектура - Легко добавлять пользовательские инструменты и провайдеры ресурсов
- Поддержка мультипространств имен - Организация инструментов и ресурсов в изолированных пространствах имен

## Примечания

Требует настройки аутентификации и включает CLI команды для управления инструментами и ресурсами. Название проекта происходит от истоков слова Guanaco — верблюдовых, обитающих в Южной Америке. Обучающее видео доступно на YouTube.
