MCP сервер, который предоставляет стандартизированный доступ к REST API OpenMetadata, позволяя взаимодействовать с каталогами данных, инструментами управления и функциями управления метаданными.

## Установка

### Smithery

```bash
npx -y @smithery/cli install @yangkyeongmo/mcp-server-openmetadata --client claude
```

### Из исходников

```bash
python -m src.main --apis table,database,databaseschema
```

## Возможности

- Полные CRUD операции для основных сущностей данных (таблицы, базы данных, схемы)
- Управление ресурсами данных (дашборды, графики, пайплайны, топики, метрики, контейнеры, отчёты, ML модели)
- Управление пользователями и командами
- Управление и классификация (классификации, глоссарии, теги)
- Системные операции (боты, сервисы, события)
- Аналитика и мониторинг (связи данных, использование, поиск)
- Управление качеством данных (тест-кейсы, тестовые наборы)
- Контроль доступа и безопасность (политики, роли)
- Управление доменами и продуктами данных
- Модульный выбор групп API

## Переменные окружения

### Обязательные
- `OPENMETADATA_HOST` - URL вашего хоста OpenMetadata

### Опциональные
- `OPENMETADATA_JWT_TOKEN` - JWT токен для аутентификации (рекомендуемый метод)
- `OPENMETADATA_USERNAME` - Имя пользователя для базовой аутентификации

## Примечания

Поддерживает выборочное включение групп API через аргументы командной строки. Реализовано более 200 API эндпоинтов OpenMetadata в 16 различных функциональных областях, включая качество данных, отслеживание связей и функции управления.
