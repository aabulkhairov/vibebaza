MCP сервер для интеграции с API Placid.app — позволяет получать списки шаблонов и генерировать изображения и видео с помощью динамического контента и шаблонов.

## Установка

### Smithery (Рекомендуется)

```bash
npx -y @smithery/cli install @felores/placid-mcp-server --client claude
```

### NPX

```bash
npx @felores/placid-mcp-server
```

## Конфигурация

### Claude Desktop/Cline

```json
{
  "mcpServers": {
    "placid": {
      "command": "npx",
      "args": ["@felores/placid-mcp-server"],
      "env": {
        "PLACID_API_TOKEN": "your-api-token"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `placid_list_templates` | Получает список доступных шаблонов Placid с возможностью фильтрации по ID коллекции, пользовательским данным или тегам |
| `placid_generate_video` | Генерирует видео, объединяя шаблоны Placid с динамическим контентом — видео, изображениями и текстом |
| `placid_generate_image` | Генерирует статические изображения, объединяя шаблоны Placid с динамическим контентом — текстом и изображениями |

## Возможности

- Получение списка доступных шаблонов Placid с опциями фильтрации
- Генерация изображений и видео с использованием шаблонов и динамического контента
- Безопасное управление API токенами
- Обработка ошибок и валидация данных
- Типобезопасная реализация

## Примечания

Требует Node.js версии 18 или выше. Получите ваш API токен в настройках Placid.app > API. Для видео длиннее 60 секунд время обработки увеличивается — вы получите ID задачи для проверки статуса в панели управления Placid.
