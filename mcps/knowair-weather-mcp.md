Комплексный MCP сервер, предоставляющий данные о погоде в реальном времени, мониторинг качества воздуха, прогнозы и астрономическую информацию на основе Caiyun Weather API с китайской локализацией.

## Установка

### uvx

```bash
uvx mcp-knowair-weather
```

### uv (macOS/Linux)

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### uv (Windows)

```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### Локальная разработка

```bash
uv run mcp-knowair-weather
```

### MCP Inspector

```bash
npx @modelcontextprotocol/inspector uv --directory /ABSOLUTE/PATH/TO/Weather-MCP run mcp-knowair-weather
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_realtime_weather` | 获取实时天气数据 - 温度、体感温度、湿度、风速、气压、能见度、空气质量、生活指数等 |
| `get_hourly_forecast` | 小时级天气预报（1-360小时）- 逐小时温度、天气现象、降水概率、风速、完整空气质量数据及趋势分析 |
| `get_daily_forecast` | 日级天气预报（1-15天）- 每日温度范围、天气现象、降水概率、风速、日出日落、生活指数等 |
| `get_air_quality_station_forecast` | 综合空气质量预报（监测站增强版）- 15天逐小时空气质量预报，智能结合监测站数据和API预报 |
| `get_minutely_precipitation` | 分钟级降水预报 - 未来2小时逐分钟降水强度、降水概率预测 |
| `get_comprehensive_weather` | 综合天气数据接口 - 实时、预报、预警、天文信息的一站式综合报告 |
| `get_weather_alerts` | 天气预警信息 - 当前生效的天气预警详情 |
| `get_astronomy_info` | 天文信息（1-15天）- 日出日落、月出月落、月相信息 |
| `get_historical_weather` | 历史天气数据 - 过去1-72小时的天气历史数据 |

## Возможности

- Данные о погоде в реальном времени - температура, ощущаемая температура, влажность, скорость ветра, видимость и другая полная метеорологическая информация
- Всесторонний мониторинг качества воздуха - PM2.5/PM10/O3/SO2/NO2/CO и стандарты AQI Китая и США
- Почасовой прогноз качества воздуха - 1-72 часа почасового прогноза тенденций AQI/PM2.5 с цветной классификацией
- Анализ тенденций качества воздуха - автоматический анализ тенденций улучшения/ухудшения качества воздуха, предоставление рекомендаций для здоровья
- Минутный прогноз осадков - прогноз интенсивности осадков на ближайшие 2 часа по минутам
- Расширенный диапазон прогнозов - прогнозы на 1-360 часов / 1-15 дней, поддержка долгосрочного планирования
- Система предупреждений о погоде - уведомления о предупреждениях в реальном времени
- Астрономическая информация - восход и закат солнца, фазы луны, время восхода и захода луны
- Исторические данные о погоде - запрос исторической погоды за последние 72 часа
- Китайская локализация - полная китайская локализация погодных явлений и индексов жизни

## Примечания

Требует координаты долготы (от -180 до 180) и широты (от -90 до 90). Поддерживает полное распознавание погодных явлений и китайский перевод. Работает на базе Caiyun Weather API с глобальными данными о погоде в реальном времени, высокоточным мониторингом качества воздуха, минутными прогнозами осадков для крупных китайских городов и официальными предупреждениями о погоде.
