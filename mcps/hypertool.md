MCP сервер, который создает сфокусированные и горячо заменяемые наборы инструментов из множества MCP серверов, чтобы уменьшить перегрузку инструментами и повысить точность выбора AI инструментов. Работает как прокси, который организует неограниченное количество инструментов в специализированные коллекции для конкретных задач.

## Установка

### NPX

```bash
npx -y @toolprint/hypertool-mcp mcp run --mcp-config .mcp.hypertool.json
```

### Режим персоны

```bash
npx -y @toolprint/hypertool-mcp mcp run --persona web-dev
```

### Команда настройки

```bash
cp .mcp.json .mcp.hypertool.json
```

## Конфигурация

### Claude Desktop/Cursor

```json
{
  "mcpServers": {
    "hypertool": {
      "command": "npx",
      "args": ["-y", "@toolprint/hypertool-mcp", "mcp", "run", "--mcp-config", ".mcp.hypertool.json"]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `list-available-tools` | Показывает оценку токенов для всех доступных инструментов на всех MCP серверах |
| `get-active-toolset` | Показывает использование токенов для вашего текущего активного набора инструментов |

## Возможности

- Подключение неограниченного количества MCP серверов без ограничения в 100 инструментов
- Создание специализированных наборов инструментов с 3-5 сфокусированными инструментами вместо 50+ хаотичных
- Горячая замена наборов инструментов динамически на основе текущей задачи
- Предварительно настроенные персоны с встроенными MCP серверами и курированными наборами инструментов
- Умные описания и аннотации инструментов для улучшения выбора AI инструментов на 89%
- Измерение контекста с оценкой токенов для каждого инструмента
- Мониторинг состояния сервера и автоматическое переподключение
- Режим конфигурации, который отделяет управление наборами инструментов от операционных инструментов
- HTTP режим для постоянных соединений и более быстрых ответов
- Группы серверов для организации и запуска связанных MCP серверов вместе

## Примечания

Наборы инструментов и конфигурации хранятся локально в ~/.toolprint/hypertool-mcp/. Работает с любым MCP-совместимым приложением. Cursor имеет полную поддержку горячей замены, Claude Desktop работает с перезапуском, горячая замена Claude Code появится в ближайшее время. Доступные персоны включают web-dev, data-scientist, devops, content-creator и researcher.
