MCP сервер, который предоставляет интерфейс на естественном языке для управления Ableton Live через устройство Max for Live, совместимый с Claude, Gemini, OpenAI и локальными AI моделями.

## Установка

### Загрузка устройства Max for Live

```bash
Download Producer_Pal.amxd from https://github.com/adamjmurray/producer-pal/releases/latest/download/Producer_Pal.amxd
```

## Возможности

- Управление Ableton Live на естественном языке
- Совместимость с Claude, Gemini, OpenAI и локальными AI моделями
- Установка простым перетаскиванием как устройство Max for Live
- Полностью открытая реализация

## Примеры использования

```
connect to ableton
```

## Ресурсы

- [GitHub Repository](https://github.com/adamjmurray/producer-pal)

## Примечания

Этот MCP сервер встроен в устройство Max for Live (.amxd файл) для простой установки. Требует следования руководству по установке для конкретных AI сервисов. Проект распространяется под лицензией MIT и полностью бесплатен навсегда. Полная документация и видеоуроки доступны на producer-pal.org.
