## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `message` | Содержимое вашего сообщения для ChatCBI |
| `chatID` | Идентифицирует разговор. Если chatID был предоставлен в запросе, это будет... |
| `message` | Ответ ChatCBI на сообщение. |
| `relatedContent` | Список связанных ссылок. |
| `sources` | Список источников, используемых для генерации ответа. |
| `suggestions` | Список предлагаемых последующих вопросов. |
| `title` | Заголовок чата |
