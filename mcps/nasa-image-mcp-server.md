MCP сервер, который предоставляет доступ к публичным API NASA для получения астрономических изображений, фотографий с марсоходов, спутниковых снимков Земли и данных о околоземных объектах со встроенными возможностями анализа изображений.

## Установка

### PyPI

```bash
pip install nasa-mcp-server
```

### UVX

```bash
uvx nasa-mcp-server
```

## Конфигурация

### VS Code

```json
{
  "servers": {
    "nasa-mcp": {
      "command": "uvx",
      "args": ["nasa-mcp-server"],
      "env": {
        "NASA_API_KEY": "YOUR_NASA_API_KEY_HERE"
      }
    }
  }
}
```

### Claude Desktop

```json
{
  "mcpServers": {
    "nasa-mcp": {
      "command": "uvx",
      "args": ["nasa-mcp-server"],
      "env": {
        "NASA_API_KEY": "YOUR_NASA_API_KEY_HERE"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get_apod` | Получение астрономических изображений дня с описаниями, поддерживает диапазоны дат и случайные изображения |
| `get_mars_image` | Доступ к фотографиям с марсоходов с различными опциями камер и фильтрацией по датам |
| `get_neo_feed` | Получение данных о околоземных объектах и информации о сближении астероидов |
| `get_earth_image_tool` | Получение спутниковых снимков Земли со спутника NASA DSCOVR с камеры EPIC |
| `get_gibs_image` | Доступ к высококачественным спутниковым снимкам Земли из Global Imagery Browse Services NASA |
| `get_gibs_layers` | Получение информации о всех доступных слоях изображений GIBS и их возможностях |
| `get_image_analyze` | Загрузка и анализ изображений по URL с автоматической обработкой и конвертацией в base64 для анализа LLM |

## Возможности

- Астрономическое изображение дня (APOD) с ежедневными астрономическими снимками и описаниями
- Изображения с марсоходов с различными опциями камер и фильтрацией по датам
- Данные о околоземных объектах (NEO) с информацией об астероидах и сближениях
- Снимки Земли (EPIC) со спутника NASA DSCOVR
- Спутниковые изображения GIBS с высококачественными снимками Земли
- Информация о слоях GIBS для доступных наборов спутниковых данных
- Инструмент анализа изображений с автоматическим определением формата, сжатием и конвертацией в base64
- Комплексная обработка ошибок для таймаутов API и некорректных параметров

## Примечания

Требует Python 3.8+ и бесплатный API ключ NASA с https://api.nasa.gov/. Инструмент анализа изображений возвращает формат imagecontent, который в настоящее время поддерживается только в Claude 4. Максимальный диапазон дат для данных NEO составляет 7 дней согласно ограничениям NASA API.
