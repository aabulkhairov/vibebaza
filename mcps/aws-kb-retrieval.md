### Установка


## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `query` | (string): Поисковый запрос для получения данных. |
| `knowledgeBaseId` | (string): ID базы знаний AWS. |
| `retrieve_from_aws_kb` | Выполнение операций получения данных с использованием базы знаний AWS. |

## Возможности

- RAG (Retrieval-Augmented Generation)**: Получение контекста из базы знаний AWS на основе запроса и ID базы знаний.
- Поддержка получения множественных результатов**: Возможность получения настраиваемого количества результатов.
