Официальный PostHog MCP сервер для взаимодействия с PostHog аналитикой, фича-флагами, отслеживанием ошибок и многим другим.

## Установка

### NPX Wizard

```bash
npx @posthog/wizard@latest mcp add
```

## Ресурсы

- [GitHub Repository](https://github.com/posthog/mcp)

## Примечания

MCP сервер был перемещен в PostHog Monorepo. Мастер автоматически устанавливает MCP сервер в Cursor, Claude, Claude Code, VS Code и Zed. Полная документация доступна по адресу https://posthog.com/docs/model-context-protocol
