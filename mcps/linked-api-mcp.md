MCP сервер, который подключает аккаунты LinkedIn к AI-ассистентам, обеспечивая автоматизированную генерацию лидов, анализ профилей, отправку сообщений и получение данных LinkedIn через облачный браузер.

## Возможности

- Автоматизация продаж - поиск лидов, анализ профилей, создание персонализированных обращений
- Помощь в рекрутинге - поиск кандидатов с определенными навыками и отправка первоначальных предложений
- Управление беседами - чтение существующих разговоров в LinkedIn и предложение ответов
- Маркетинговые исследования - сбор данных о конкурентах, информации о компаниях и отраслевой аналитики
- Поиск и анализ профилей LinkedIn
- Автоматизированная отправка сообщений в LinkedIn
- Генерация и квалификация лидов

## Примеры использования

```
Search for software engineers at companies with 50-200 employees in San Francisco
```

```
Find leads and check their profiles
```

```

Review candidate experience and send initial outreach
```

## Примечания

Этот MCP сервер использует облачный браузер для безопасного и автоматического выполнения операций в LinkedIn. Руководства по установке, документация по доступным инструментам и примеры использования доступны на сайте linkedapi.io. Лицензия MIT.
