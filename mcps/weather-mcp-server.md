MCP сервер, который предоставляет всесторонние метеорологические данные, включая текущие условия, прогнозы, исторические данные, качество воздуха, астрономическую информацию и погодные предупреждения с использованием WeatherAPI.

## Установка

### Smithery

```bash
npx -y @smithery/cli install @devilcoder01/weather-mcp-server --client claude
```

### Из исходного кода

```bash
git clone https://github.com/yourusername/Weather_mcp_server.git
cd Weather_mcp_server
uv venv
uv pip install -e .
```

## Возможности

- Текущие погодные условия
- Прогнозы погоды (1-14 дней)
- Исторические данные о погоде
- Погодные предупреждения
- Информация о качестве воздуха
- Астрономические данные (восход, закат, фазы луны)
- Поиск местоположений
- Информация о временных зонах
- Спортивные события

## Переменные окружения

### Обязательные
- `WEATHER_API_KEY` - ключ WeatherAPI для доступа к метеорологическим данным

## Ресурсы

- [GitHub Repository](https://github.com/devilcoder01/weather-mcp-server)

## Примечания

Требует Python 3.13+ и менеджер пакетов uv. Создан с помощью FastAPI и MCP фреймворка. Сервер по умолчанию запускается на http://localhost:8000.
