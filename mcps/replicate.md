Реализация Model Context Protocol сервера для Replicate, которая позволяет искать, запускать и управлять моделями машинного обучения через простой интерфейс на основе инструментов.

## Установка

### NPM Global

```bash
npm install -g mcp-replicate
```

### Из исходного кода

```bash
git clone https://github.com/deepfates/mcp-replicate
cd mcp-replicate
npm install
npm run build
npm start
```

### NPX

```bash
npx mcp-replicate
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "replicate": {
      "command": "mcp-replicate",
      "env": {
        "REPLICATE_API_TOKEN": "your_token_here"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `search_models` | Поиск моделей с помощью семантического поиска |
| `list_models` | Просмотр доступных моделей |
| `get_model` | Получение подробной информации о конкретной модели |
| `list_collections` | Просмотр коллекций моделей |
| `get_collection` | Получение подробной информации о конкретной коллекции |
| `create_prediction` | Запуск модели с вашими входными данными |
| `create_and_poll_prediction` | Запуск модели с вашими входными данными и ожидание завершения |
| `get_prediction` | Проверка статуса предсказания |
| `cancel_prediction` | Остановка запущенного предсказания |
| `list_predictions` | Просмотр ваших недавних предсказаний |
| `view_image` | Открытие изображения в браузере |
| `clear_image_cache` | Очистка кэшированных изображений |
| `get_image_cache_stats` | Проверка использования кэша |

## Возможности

- Поиск моделей с помощью семантического поиска
- Просмотр моделей и коллекций
- Получение подробной информации о моделях и версиях
- Создание предсказаний с текстовым или структурированным вводом
- Отслеживание статуса предсказаний
- Отмена запущенных предсказаний
- Просмотр ваших недавних предсказаний
- Просмотр сгенерированных изображений в браузере
- Управление кэшем изображений для лучшей производительности
