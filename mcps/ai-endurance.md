Подключение платформы AI Endurance к Claude для разговорного доступа к тренировочным данным, тренировкам, аналитике производительности и управлению тренировочными планами для велоспорта, бега и плавания.

## Конфигурация

### Claude.ai

```json
Name: AI Endurance
Remote MCP Server URL: https://aiendurance.com/mcp
```

### SSE Transport конфигурация

```json
{
  "url": "https://aiendurance.com/mcp",
  "transport": {
    "type": "sse"
  },
  "auth": {
    "type": "oauth",
    "clientUrl": "https://aiendurance.com/authorize/",
    "authorizationUrl": "https://aiendurance.com/api/o/token/",
    "scope": "read write"
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `getUser` | Просмотр профиля включая тренировочные зоны, тип пользователя, единицы измерения и настройки |
| `setZones` | Обновление тренировочных зон для велоспорта (мощность) или бега (темп/мощность) |
| `getAvailability` | Просмотр еженедельных часов тренировок и расписания доступности по дням для каждого типа активности |
| `getPlannedWorkouts` | Получение запланированных тренировок на период (по умолчанию: следующие 14 дней) |
| `changeWorkoutDate` | Перенос тренировки на другую дату и синхронизация с подключёнными платформами |
| `skipWorkout` | Удаление тренировки из плана |
| `markWorkout` | Отметить тренировку как выполненную или невыполненную |
| `changeWorkoutAdvice` | Добавить или обновить тренерский совет для конкретной тренировки |
| `createRideRunWorkout` | Создание кастомной структурированной тренировки для велоспорта или бега с интервалами |
| `createSwimWorkout` | Создание кастомной тренировки по плаванию со структурированными секциями и сетами |
| `getCyclingActivity` | Список последних велотренировок с summary метриками |
| `getRunningActivity` | Список последних беговых тренировок с summary метриками |
| `getSwimmingActivity` | Список последних тренировок по плаванию с summary метриками |
| `getCyclingActivityDetail` | Детальные метрики конкретной велотренировки включая time-series данные |
| `getRunningActivityDetail` | Детальные метрики конкретной беговой тренировки включая time-series данные |

## Возможности

- Управление тренировочным планом — просмотр, изменение и создание тренировок со структурированными интервалами
- Анализ активностей — доступ к детальным метрикам велоспорта, бега и плавания
- Прогнозы производительности — ML-предсказания времени на гонках и прогнозирование фитнеса
- Отслеживание восстановления — мониторинг HRV, пульса в покое и готовности к тренировкам
- Управление зонами — обновление и просмотр тренировочных зон (темп, мощность)
- Планирование тренировок — перенос тренировок, настройка доступности, отслеживание прогресса плана
- Гоночные цели — управление основными и второстепенными гоночными целями

## Примечания

Требуется аккаунт AI Endurance и активная подписка или бесплатный trial. Поддерживает OAuth 2.0 аутентификацию со scopes read/write. Совместим с Claude Desktop, Cursor, Continue, Cline и другими MCP клиентами. Синхронизируется с Garmin, TrainingPeaks, Zwift и другими платформами.
