MCP сервер, который позволяет Claude получать данные о недавних и исторических играх NBA и статистике, используя открытый nba_api.

## Установка

### Из исходного кода

```bash
uv venv
.venv\Scripts\activate
uv pip install -e .
```

## Возможности

- Получение финального счета для всех игр, которые прошли вчера / в прошлом
- Получение базовой разбивки очков/подборов/передач для всех игроков, которые играли в играх вчера / в прошлом
- Получение полной статистики PTS/REB/AST/STL/BLK/TO/PLUS_MINUS/MIN для игр, которые прошли вчера / в прошлом
- Получение четырех факторов для всех игр, которые прошли вчера / в прошлом

## Ресурсы

- [GitHub Repository](https://github.com/Taidgh-Robinson/nba-mcp-server)

## Примечания

Использует открытый Python пакет nba_api. Требует добавления конфигурации в настройки Claude, как и любой другой MCP сервер.
