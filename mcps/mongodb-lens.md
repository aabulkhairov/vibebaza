Локальный Model Context Protocol (MCP) сервер, предоставляющий полнофункциональный доступ к базам данных MongoDB через естественный язык с помощью LLM для выполнения запросов, агрегаций, оптимизации производительности и управления операциями базы данных.

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `add-connection-alias` | Добавить новый псевдоним соединения MongoDB |
| `aggregate-data` | Выполнить конвейеры агрегации |
| `analyze-query-patterns` | Анализировать живые запросы и предлагать оптимизации |
| `analyze-schema` | Автоматически выводить схемы коллекций |
| `bulk-operations` | Выполнять множественные операции эффективно (требует подтверждения для деструктивных операций) |
| `clear-cache` | Очистить кэши памяти для обеспечения свежих данных |
| `collation-query` | Найти документы с языково-специфичными правилами сопоставления |
| `compare-schemas` | Сравнить схемы между двумя коллекциями |
| `connect-mongodb` | Подключиться к другому MongoDB URI |
| `connect-original` | Подключиться обратно к оригинальному MongoDB URI, использованному при запуске |
| `count-documents` | Подсчитать документы, соответствующие указанным критериям |
| `create-collection` | Создать новые коллекции с пользовательскими опциями |
| `create-database` | Создать новую базу данных с опцией переключения на неё |
| `create-index` | Создать новые индексы для оптимизации производительности |
| `create-timeseries` | Создать коллекции временных рядов для темпоральных данных |

## Возможности

- Запросы на естественном языке для баз данных MongoDB
- Выполнение запросов с фильтрами, проекциями и сортировкой
- Выполнение конвейеров агрегации
- Анализ и вывод схем
- Оптимизация производительности и анализ запросов
- Управление индексами и рекомендации
- Управление базами данных и коллекциями
- Управление пользователями и ролями
- CRUD операции с подтверждениями безопасности
- Управление файлами GridFS

## Примечания

Включает комплексную защиту данных с требованиями подтверждения для деструктивных операций. Содержит более 35 инструментов, 12 ресурсов и 13 специализированных промптов для операций MongoDB. Поддерживает методы установки Docker и NPM, а также имеет руководства по настройке клиентов для Claude Desktop и Cursor.
