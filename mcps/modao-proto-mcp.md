AI-сервер для генерации HTML прототипов, который преобразует описания на естественном языке в полноценный HTML код с современным дизайном и адаптивной версткой, интегрированный с рабочим пространством Modao.

## Установка

### Установка через NPM

```bash
npm install
```

### Сборка

```bash
npm run build
```

### Запуск сервера

```bash
node dist/index.js --token YOUR_API_TOKEN
```

### Запуск с кастомным URL

```bash
node dist/index.js --token YOUR_API_TOKEN --url https://modao.cc
```

### Запуск с отладкой

```bash
node dist/index.js --token YOUR_API_TOKEN --debug
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `gen_html` | Генерирует полноценный HTML код на основе описаний пользователя, поддерживает современный дизайн и адаптивную верстку |
| `gen_description` | Генерирует подробные документы с техническими требованиями дизайна на основе кратких требований пользователя |
| `import_html` | Импортирует сгенерированный HTML в персональное рабочее пространство пользователя, используя ключ из gen_html |

## Возможности

- Генерация HTML кода с современным дизайном и адаптивной версткой
- Генерация описаний дизайна для подробных спецификаций
- Импорт HTML в персональное рабочее пространство
- Полная совместимость с Model Context Protocol
- Расширяемая архитектура для добавления новых инструментов
- Множественные форматы параметров и обработка ошибок

## Переменные окружения

### Обязательные
- `token` - токен доступа к API сервису (обязательный)

### Опциональные
- `url` - URL API сервиса (опциональный, по умолчанию https://modao.cc)
- `debug` - включить режим отладки (опциональный)

## Примечания

Требует токен от modao.cc (войти → аватар → настройки токена → создать токен). Если gen_html не работает из-за недостатка кредитов, пополните баланс на modao.cc/ai. Поддерживает как базовый рабочий процесс (генерация → импорт), так и расширенный рабочий процесс (описание → генерация → импорт).
