MCP сервер для доступа к базе данных лекарственно-генных взаимодействий (DGIdb), который позволяет пользователям запрашивать взаимодействия лекарство-ген, атрибуты лекарств и генетическую информацию через Claude Desktop.

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "dgidb": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://dgidb-mcp-server.quentincody.workers.dev/sse"
      ]
    }
  }
}
```

## Возможности

- Запросы лекарственно-генных взаимодействий из DGIdb
- Доступ к комплексной базе данных из множества источников
- Запросы на естественном языке через Claude Desktop
- Интеграция с DGIdb GraphQL API
- Поддержка деплоя на Cloudflare Workers

## Примеры использования

```
Какие взаимодействия есть у препарата Иматиниб?
```

```
Найди гены, которые взаимодействуют с Довитинибом.
```

```
Покажи мне атрибуты препарата Траметиниб.
```

## Примечания

Сервер использует публично доступный DGIdb GraphQL API по адресу https://dgidb.org/api/graphql. Доступен под лицензией MIT с требованием академического цитирования - академические/исследовательские публикации должны содержать соответствующую ссылку. Может быть развернут на Cloudflare Workers и подключен к другим MCP клиентам, таким как AI Playground.
