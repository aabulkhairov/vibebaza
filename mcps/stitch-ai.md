MCP сервер для системы управления памятью Stitch AI, который предоставляет инструменты для создания, извлечения и управления памятью AI агентов в децентрализованных пространствах знаний.

## Установка

### Из исходного кода

```bash
git clone https://github.com/StitchAI/stitch-ai-mcp.git
npm install @modelcontextprotocol/sdk zod
npm install -D @types/node typescript
npm run start
```

## Конфигурация

### Claude Desktop

```json
{
    "mcpServers": {
        "stitchai": {
            "command": "npx",
            "args": [
                "ts-node",
                "/path/to/cloned/stitch-ai-mcp/src/server.ts"
            ],
            "env": {
                "API_KEY": "<STITCH_AI_API_KEY>",
                "BASE_URL": "https://api-demo.stitch-ai.co"
            }
        }
    }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `create_space` | Создает новое пространство памяти с указанным именем и типом |
| `delete_space` | Удаляет пространство памяти с указанным именем |
| `get_all_spaces` | Получает список всех доступных пространств памяти |
| `upload_memory` | Загружает новую память в указанное пространство памяти |
| `get_memory` | Извлекает конкретную память по ID из пространства памяти |
| `get_all_memories` | Извлекает всю память из указанного пространства памяти с опциональной фильтрацией и пагинацией |

## Возможности

- Создание и управление пространствами памяти для AI агентов
- Загрузка и хранение содержимого памяти с сообщениями
- Извлечение конкретной памяти по ID
- Просмотр всех доступных пространств памяти
- Фильтрация памяти по именам с поддержкой пагинации
- Удаление пространств памяти, когда они больше не нужны

## Переменные окружения

### Обязательные
- `API_KEY` - API ключ Stitch AI для аутентификации

### Опциональные
- `BASE_URL` - Базовый URL для Stitch AI API (по умолчанию https://api-demo.stitch-ai.co)

## Примечания

Это децентрализованный хаб знаний для AI, который требует API ключ Stitch AI. Сервер по умолчанию подключается к демо-среде Stitch AI, но может быть настроен для других сред.
