Полная интеграция n8n API для Claude Desktop и Cursor с 41 инструментом для управления рабочими процессами, автоматизации задач и контроля всех аспектов n8n напрямую через разговоры с AI. Включает 100+ готовых шаблонов рабочих процессов и операции, оптимизированные по токенам.

## Установка

### NPM Global

```bash
npm install -g mcp-n8n
```

### NPX

```bash
npx -y mcp-n8n
```

### Из исходного кода

```bash
git clone https://github.com/gomakers-ai/mcp-n8n.git
cd mcp-n8n
npm install
npm run build
```

## Конфигурация

### Claude Desktop (глобальная установка)

```json
{
  "mcpServers": {
    "n8n": {
      "command": "mcp-n8n",
      "env": {
        "N8N_BASE_URL": "https://your-n8n-instance.com",
        "N8N_API_KEY": "your-api-key-here"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `n8n_create_workflow` | Создание новых рабочих процессов |
| `n8n_list_workflows_summary` | Токен-эффективный список (только id, имя, активность, теги) |
| `n8n_list_workflows` | Список с полными деталями и опциональной фильтрацией полей |
| `n8n_get_workflow` | Получение детальной информации о рабочем процессе |
| `n8n_update_workflow` | Изменение существующих рабочих процессов |
| `n8n_delete_workflow` | Окончательное удаление рабочих процессов |
| `n8n_activate_workflow` | Включение выполнения рабочего процесса |
| `n8n_deactivate_workflow` | Приостановка выполнения рабочего процесса |
| `n8n_transfer_workflow` | Перемещение между проектами |
| `n8n_get_workflow_tags` | Просмотр тегов рабочего процесса |
| `n8n_update_workflow_tags` | Изменение тегов рабочего процесса |
| `n8n_list_workflow_templates` | Просмотр доступных шаблонов |
| `n8n_get_workflow_template` | Просмотр деталей шаблона |
| `n8n_create_workflow_from_template` | Создание из шаблона |
| `n8n_list_executions` | Фильтрация по статусу, рабочему процессу, проекту |

## Возможности

- 90% сокращение токенов для списка рабочих процессов с оптимизированными эндпоинтами
- Фильтрация полей - запрашивайте только нужные данные
- Умные настройки по умолчанию - сокращение с 100 до 10-20 результатов на запрос
- Создание и деплой рабочих процессов с описанием на естественном языке
- Полные CRUD операции для управления жизненным циклом рабочих процессов
- Контроль активации рабочих процессов
- Мониторинг выполнения в реальном времени с продвинутыми фильтрами
- Восстановление после ошибок с автоматическими повторными попытками
- Безопасное управление учетными данными с обнаружением схем
- 100 готовых к продакшену шаблонов рабочих процессов в 13 категориях

## Переменные окружения

### Обязательные
- `N8N_BASE_URL` - URL вашего экземпляра n8n
- `N8N_API_KEY` - API ключ n8n для аутентификации

## Примечания

Требует платный план n8n Cloud или самостоятельно размещенный n8n для доступа к API. Совместим с n8n v1.0.0+. Некоторые операции требуют прав владельца/администратора. Сервер специально оптимизирован для минимизации потребления токенов со специализированными эндпоинтами сводок и интеллектуальными предупреждениями.
