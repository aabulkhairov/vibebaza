Сервис для создания красивых бейджей для проектов MCP (Model Context Protocol), которые подчеркивают их функциональность и возможности.

## Возможности

- Четыре типа бейджей: стандартный, сервер, клиент и разработческий
- Поддержка параметра статуса для стандартных бейджей (включен/выключен)
- Поддержка параметра возможностей для серверных и клиентских бейджей
- Поддержка функций MCP сервера: ресурсы, промпты, инструменты, семплирование
- Поддержка функций MCP клиента: ресурсы, промпты, инструменты, семплирование, корни
- Специальные бейджи для проектов, связанных с разработкой MCP
- Доступны в форматах Markdown и HTML img тегов

## Ресурсы

- [GitHub Repository](https://github.com/mcpx-dev/mcp-badges)

## Примечания

Это сервис бейджей, а не MCP сервер как таковой. Бейджи генерируются через URL параметры на https://badge.mcpx.dev с различными типами и спецификациями возможностей.
