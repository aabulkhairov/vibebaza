Open-source MCP сервер, который соединяет AI агентов с корпоративными системами, обеспечивая безопасный доступ к бизнес-приложениям, данным производства в реальном времени, системам управления документами и пользовательским AI пайплайнам через развертывание в Azure с интеграцией Private Link и Key Vault.

## Возможности

- Нативная поддержка Microsoft Copilot Studio через MCP (Preview)
- Встроенная совместимость с Anthropic Claude через десктопное приложение
- Поддержка OpenAPI для платформ вроде Dify
- Деплой в Azure со встроенной поддержкой Private Link
- Интеграция с Key Vault для безопасного управления секретами
- Интеграция с Azure AD для корпоративной аутентификации
- Стриминг данных сенсоров и метрик в реальном времени
- Контроль доступа на основе ролей с логированием аудита
- Универсальная связность с базами данных
- Стриминг данных в реальном времени с событийно-ориентированной архитектурой

## Ресурсы

- [GitHub Repository](https://github.com/spgoodman/createveai-nexus-server)

## Примечания

Поддержка MCP в Microsoft Copilot Studio сейчас находится в preview. Сервер спроектирован для корпоративного развертывания с комплексными контролями безопасности, а RootUK предоставляет профессиональные услуги интеграции для корпоративных клиентов. Совместим с автоматизацией workflow в n8n и включает обширную документацию по настройке, конфигурации и пользовательским интеграциям.
