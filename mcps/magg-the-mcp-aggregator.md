Мета-MCP сервер, который работает как универсальный хаб, позволяя LLM автономно обнаруживать, устанавливать и оркестрировать множественные MCP серверы - по сути предоставляя AI ассистентам возможность расширять собственные способности по требованию. Включает `mbro`, мощный CLI браузер MCP серверов с возможностями скриптинга
