Мета-MCP сервер, который выступает в роли сервиса для поиска и предоставления инструментов, предоставляя доступ к обширной базе данных из 2,037+ MCP серверов из множества курируемых источников, помогая AI-ассистентам находить подходящие инструменты.

## Установка

### uvx

```bash
uvx mcp-mcp
```

### Из исходного кода

```bash
git clone https://github.com/your-username/mcp-mcp.git
cd mcp-mcp
uv sync
uv run main.py
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "mcp-mcp": {
      "command": "uvx",
      "args": ["mcp-mcp"]
    }
  }
}
```

### Claude Desktop (pipx)

```json
{
  "mcpServers": {
    "mcp-mcp": {
      "command": "mcp-mcp"
    }
  }
}
```

## Возможности

- Доступ к 2,037+ MCP серверам из множества курируемых источников
- Агрегация официальных MCP серверов (modelcontextprotocol/servers)
- Коллекции сообщества от Punkpeye & Appcypher awesome-листов
- Интеллектуальная дедупликация исключает дубликаты между источниками
- Автоматические обновления каждые 3 часа с новейшими серверами сообщества
- Семантический поиск с предвычисленными эмбеддингами для ответов менее чем за секунду
- Усиленная безопасность с мидлваром валидации источников

## Примечания

Сервер поддерживает режимы транспорта как STDIO, так и HTTP. Файлы конфигурации расположены в ~/Library/Application Support/Claude/claude_desktop_config.json на macOS и %APPDATA%\Claude\claude_desktop_config.json на Windows. База данных автоматически обновляется каждые 3 часа.
