Универсальный MCP сервер для интерактивного исследования CSV датасетов, который работает как персональный ассистент Data Scientist и превращает сложные наборы данных в понятные и практичные инсайты.

## Установка

### Скрипт установки

```bash
python setup.py
```

### Сборка для разработки

```bash
uv sync
uv build
```

### Установка через PyPI

```bash
uvx mcp-server-ds
```

## Конфигурация

### Claude Desktop - Разработка

```json
{
  "mcpServers": {
    "mcp-server-ds": {
      "command": "uv",
      "args": [
        "--directory",
        "/Users/username/src/mcp-server-ds",
        "run",
        "mcp-server-ds"
      ]
    }
  }
}
```

### Claude Desktop - Релизная версия

```json
{
  "mcpServers": {
    "mcp-server-ds": {
      "command": "uvx",
      "args": [
        "mcp-server-ds"
      ]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `load-csv` | Загружает CSV файл в DataFrame с возможностью кастомного именования |
| `run-script` | Выполняет Python скрипты для анализа данных и визуализации |

## Возможности

- Интерактивное исследование CSV датасетов
- Автоматическая генерация инсайтов с минимальным участием человека
- Поддержка больших датасетов (2M+ записей)
- Анализ погодных паттернов
- Анализ данных по недвижимости
- Анализ трендов температуры и влажности
- Визуализация ветровых паттернов
- Выполнение кастомных Python скриптов
- Гибкое именование и управление DataFrame

## Примечания

ВНИМАНИЕ: Этот сервер выполняет произвольный Python код на вашей машине - используйте с осторожностью! Требует Claude Desktop и работает с шаблоном промпта 'explore-data'. Поддерживает очень большие датасеты (тестировался с 2M+ записями). Файлы конфигурации находятся в ~/Library/Application Support/Claude/claude_desktop_config.json на macOS и %APPDATA%/Claude/claude_desktop_config.json на Windows.
