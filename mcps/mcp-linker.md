Кросс-платформенный GUI инструмент для настройки в один клик и управления MCP серверами в различных AI клиентах как Claude Desktop, Cursor, VS Code и Windsurf без необходимости вызовов LLM.

## Установка

### Прямая загрузка

```bash
Download the latest release from https://github.com/milisp/mcp-linker/releases
```

## Возможности

- Локальная синхронизация конфигурации MCP серверов между несколькими AI клиентами
- Встроенный маркетплейс с более чем 600 курированными MCP серверами
- Поддержка нескольких клиентов для Claude Desktop/Code, Cursor, VS Code, Cline, Windsurf
- Кросс-платформенная поддержка для macOS, Windows, Linux (~6MB)
- Умное обнаружение Python, Node.js, uv окружений
- Опциональная зашифрованная облачная синхронизация между устройствами (Pro)
- Встроенный MCP чат с AI Finder/проводником и функциональностью блокнота

## Ресурсы

## Примечания

Для пользователей macOS, сталкивающихся с сообщением 'App is damaged', выполните: xattr -d com.apple.quarantine /path/to/MCPLinker.app или перейдите в System Preferences → Security & Privacy → Нажмите 'Open Anyway'. Pro планы доступны для функций облачной синхронизации от цены кофе в месяц.
