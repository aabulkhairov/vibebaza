MCP сервер, предоставляющий комплексные возможности для исследования рынка и бизнес-анализа через 28 инструментов, 15 бизнес-шаблонов и интеграцию с 8 источниками экономических данных, включая Alpha Vantage, BLS, Census, FRED, IMF, Nasdaq Data Link, OECD и World Bank.

## Установка

### Из исходного кода

```bash
git clone https://github.com/your-username/TAM-MCP-Server.git
cd TAM-MCP-Server
npm install
cp .env.example .env
# Edit .env with your API keys
npm run build
npm run start:http
```

### Настройка для разработки

```bash
chmod +x scripts/dev-setup.sh
./scripts/dev-setup.sh
```

### Docker

```bash
docker build -t tam-mcp-server .
docker run -p 3000:3000 --env-file .env tam-mcp-server
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `alphaVantage_getCompanyOverview` | Получить детальный обзор компании и финансовые данные |
| `alphaVantage_searchSymbols` | Поиск биржевых символов и названий компаний |
| `bls_getSeriesData` | Получить данные о занятости от Бюро статистики труда |
| `census_fetchIndustryData` | Доступ к отраслевой статистике Бюро переписи США |
| `census_fetchMarketSize` | Получить данные о размере рынка из опросов переписи |
| `fred_getSeriesObservations` | Получить временные ряды экономических данных Федерального резерва |
| `imf_getDataset` | Доступ к наборам данных Международного валютного фонда |
| `nasdaq_getDatasetTimeSeries` | Получить временные ряды Nasdaq Data Link |
| `oecd_getDataset` | Доступ к статистическим наборам данных OECD |
| `worldBank_getIndicatorData` | Получить индикаторы развития Всемирного банка |
| `industry_analysis` | Улучшенный многоисточниковый анализ отрасли с интеллектуальным ранжированием |
| `tam_analysis` | Продвинутые расчеты общего адресуемого рынка с прогнозами сценариев |
| `sam_calculator` | Обслуживаемый адресуемый рынок с анализом ограничений |
| `market_size` | Продвинутая оценка и анализ размера рынка с оценкой достоверности |
| `market_forecasting` | Прогнозирование временных рядов с анализом сценариев |

## Возможности

- 28 комплексных инструментов для анализа рынка и доступа к данным
- 15 профессиональных шаблонов бизнес-анализа для финансирования, стратегии и исследований
- Интеграция с 8 источниками экономических данных (Alpha Vantage, BLS, Census, FRED, IMF, Nasdaq Data Link, OECD, World Bank)
- Умные настройки по умолчанию для быстрого старта - не требуется исследование параметров
- Множественные транспортные протоколы (STDIO, HTTP, SSE)
- Уведомления в реальном времени для рыночной аналитики и мониторинга источников данных
- Расчеты общего адресуемого рынка (TAM) и обслуживаемого адресуемого рынка (SAM)
- Оценка размера рынка и прогнозирование с анализом сценариев
- Отраслевой анализ и конкурентная разведка
- Кросс-валидация данных и оценка качества

## Переменные окружения

### Опциональные
- `PORT` - Конфигурация порта сервера
- `NODE_ENV` - Окружение Node (development/production)
- `LOG_LEVEL` - Конфигурация уровня логирования
- `ALPHA_VANTAGE_API_KEY` - API ключ Alpha Vantage для финансовых данных компаний и акций
- `CENSUS_API_KEY` - API ключ Бюро переписи для отраслевых и демографических данных
- `FRED_API_KEY` - API ключ Federal Reserve Economic Data
- `NASDAQ_DATA_LINK_API_KEY` - API ключ Nasdaq Data Link для финансовых наборов данных
- `BLS_API_KEY` - API ключ Бюро статистики труда для повышенных лимитов

## Примечания

Сервер предоставляет умные настройки по умолчанию для всех 28 инструментов, позволяя немедленно выполнять профессиональный анализ рынка без исследования параметров. Инструменты работают с ограниченными данными даже без API ключей, хотя ключи предоставляют расширенные возможности. Поддерживает три транспортных метода с рекомендацией HTTP для продакшена.
