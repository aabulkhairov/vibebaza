Расширяемый MCP сервер, который обеспечивает совместимость с любым Python приложением командной строки, поддерживая популярные фреймворки как Click, Typer и Argparse.

## Установка

### pip

```bash
pip install pycli-mcp
```

## Возможности

- Совместим с любым Python приложением командной строки
- Поддерживает фреймворк Click
- Поддерживает фреймворк Typer
- Экспериментальная поддержка Argparse
- Расширяемая архитектура
