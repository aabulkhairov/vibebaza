Реализация MCP сервера для криптовалютной биржи Upbit, которая предоставляет инструменты для взаимодействия с сервисами биржи Upbit, включая получение рыночных данных, управление аккаунтом, создание/отмену ордеров и технический анализ.

## Установка

### Из исходного кода

```bash
git clone https://github.com/solangii/upbit-mcp-server.git
cd upbit-mcp-server
uv sync
```

### Установка UV

```bash
curl -Ls https://astral.sh/uv/install.sh | sh
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### FastMCP

```bash
fastmcp install main.py --name "Upbit API"
```

### Прямой запуск через Python

```bash
uv run python main.py
```

### Режим разработки

```bash
fastmcp dev main.py
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "upbit-mcp-server": {
      "command": "/full/path/to/upbit-mcp-server/.venv/bin/python",
      "args": [
        "/full/path/to/upbit-mcp-server/main.py"
      ]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `get_ticker` | 현재 암호화폐 시세 조회 (Получение текущих цен криптовалют) |
| `get_orderbook` | 호가창 정보 조회 (Получение информации из стакана заявок) |
| `get_trades` | 최근 체결 내역 조회 (Получение истории последних сделок) |
| `get_market_summary` | 주요 암호화폐 시장 요약 정보 확인 (Проверка сводной информации по основным криптовалютным рынкам) |
| `get_accounts` | 보유 중인 자산 목록 및 잔고 확인 (Проверка списка активов и балансов) |
| `get_orders` | 주문 내역 조회 (Получение истории ордеров) |
| `get_order` | 특정 주문 상세 정보 조회 (Получение детальной информации по конкретному ордеру) |
| `get_deposits_withdrawals` | 입출금 내역 조회 (Получение истории депозитов и выводов) |
| `create_order` | 지정가/시장가 매수/매도 주문 생성 (Создание лимитных/рыночных ордеров на покупку/продажу) |
| `cancel_order` | 주문 취소 (Отмена ордеров) |

## Возможности

- Получение рыночных данных (тикер, стакан заявок, сделки, свечные данные)
- Информация об аккаунте (баланс, история ордеров)
- Создание и отмена ордеров
- Функции депозита и вывода средств
- Инструменты технического анализа

## Переменные окружения

### Обязательные
- `UPBIT_ACCESS_KEY` - Ваш ключ доступа к API Upbit для аутентификации
- `UPBIT_SECRET_KEY` - Ваш секретный ключ API Upbit для аутентификации

## Примечания

Этот сервер может обрабатывать реальные торговые операции, поэтому используйте его осторожно. Держите ваши API ключи в безопасности и никогда не коммитьте их в публичные репозитории. Вам необходимо создать аккаунт на Upbit и получить API ключи в центре разработчиков Upbit с соответствующими разрешениями (чтение, торговля, вывод средств по мере необходимости).
