Кураторский хаб MCP серверов и ресурсов для создания AI агентов, которые взаимодействуют с реальными API на Azure, поддерживающий разработчиков на C#, Python, Java и JavaScript.

## Возможности

- Хостинг MCP серверов с использованием Azure Functions на нескольких языках (C#, Python, TypeScript)
- Официальные MCP SDK для C#, Python, TypeScript и Java
- Интеграции с AI фреймворками (OpenAI Agents SDK, Semantic Kernel, LangChain.js, Spring AI, Azure AI Agents)
- Готовые MCP серверы для данных API (Redis, PostgreSQL, MySQL, MongoDB, Azure Data Explorer)
- Серверы для разработки и инфраструктуры API (Azure CLI, Kubernetes, GitHub, Azure DevOps)
- Интеграция с сервисами Azure (Redis, PostgreSQL, MySQL, Cosmos DB, AKS)

## Ресурсы

- [GitHub Repository](https://github.com/Azure-Samples/mcp)

## Примечания

Это хаб/коллекция репозиториев, которая ссылается на различные MCP серверы и ресурсы, а не единая реализация MCP сервера. Поддерживаемые сообществом серверы предоставляются как есть без проверки или поддержки Microsoft.
