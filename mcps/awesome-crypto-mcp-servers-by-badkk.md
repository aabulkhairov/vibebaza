Подобранная коллекция крутых серверов Model Context Protocol (MCP), специально сфокусированных на функциональности криптовалют и блокчейна.

## Возможности

- Подобранная коллекция MCP серверов с фокусом на крипто
- Доступ к данным в сети, включая ERC20 токены и историю транзакций
- Бесплатные переводы USDC в сети Base с использованием технологии MPC кошелька Coinbase CDP
- Данные рынка криптовалют от CoinMarketCap
- Данные индекса страха и жадности в реальном времени и исторические данные
- Свежие новости о криптовалютах от CryptoPanic
- Данные в реальном времени для 5+ миллионов токенов в 20+ блокчейн сетях
- Возможности отслеживания транзакций китов
- Web3 AI агенты для анализа блокчейна и безопасности смарт-контрактов
- Поддержка 30+ EVM сетей с нативными токенами, ERC20, поддержкой NFT

## Примечания

Это не единственный MCP сервер, а скорее подобранный каталог связанных с крипто MCP серверов. Отдельные серверы в списке включают Bankless Onchain, mcp-free-usdc-transfer, coin_api_mcp, crypto-feargreed-mcp, cryptopanic-mcp-server, Dappier, DexPaprika, whale-tracker-mcp, Heurist Mesh Agent, EVM MCP Server, GOAT и Solana Agent Kit. Репозиторий создан Magnet Labs и включает туториалы по использованию MCP инструментов и десктопное приложение для управления MCP серверами.
