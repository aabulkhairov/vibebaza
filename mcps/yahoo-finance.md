MCP сервер (Model Context Protocol), который позволяет вашему AI взаимодействовать с Yahoo Finance для получения комплексных данных о фондовом рынке, новостях, финансовых показателях, истории цен, цепочках опционов и данных о доходах.

## Установка

### uvx (рекомендуется)

```bash
uvx yahoo-finance-server
```

### pip

```bash
pip install yahoo-finance-server
```

### Из исходников

```bash
git clone https://github.com/AgentX-ai/AgentX-mcp-servers.git
cd AgentX-mcp-servers/yahoo_finance_server
pip install -e .
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `get-ticker-info` | Получение комплексной информации об акциях, включая данные о компании, финансовые показатели и торговые метрики |
| `get-ticker-news` | Получение свежих новостных статей по символу акции |
| `search` | Поиск акций, ETF и других финансовых инструментов со связанными новостями |
| `get-top-entities` | Получение топовых компаний в секторе (компании, ETF, взаимные фонды) |
| `get-price-history` | Получение исторических данных о ценах с настраиваемыми периодами и интервалами |
| `ticker-option-chain` | Получение данных цепочки опционов для акций, включая call и put опционы |
| `ticker-earning` | Получение данных о доходах, включая исторические и предстоящие отчеты о прибыли |

## Возможности

- Данные об акциях - получение комплексной информации о тикерах, включая детали компании, финансовые показатели и торговые метрики
- Новостная лента - получение свежих новостных статей по конкретным символам акций
- Поиск - поиск акций, ETF и других финансовых инструментов с результатами новостей
- Топовые компании - получение лучших компаний, ETF и взаимных фондов по секторам
- История цен - получение исторических данных о ценах с настраиваемыми периодами и интервалами
- Цепочка опционов - получение данных цепочки опционов для акций, включая call и put опционы
- Прибыль - доступ к данным о доходах, включая предстоящие даты отчетов о прибыли
- Поддержка прокси - работа с HTTP/HTTPS/SOCKS прокси
- Быстро и асинхронно - неблокирующие операции с использованием asyncio

## Переменные окружения

### Опциональные
- `PROXY_URL` - URL прокси сервера для лучшей надежности и избежания ограничений по скорости запросов (поддерживает HTTP/HTTPS/SOCKS)

## Примечания

Требует Python 3.11 или выше. Рекомендуется настройка прокси для лучшей надежности и избежания ограничений по скорости запросов. Поддерживает различные сектора, включая технологии, здравоохранение, финансовые услуги и многие другие.
