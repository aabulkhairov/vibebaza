Node.js сервер, реализующий Model Context Protocol (MCP) для CLI приложения git-mob, обеспечивающий автоматическое управление соавторами в git коммитах во время парного программирования и моб-сессий.

## Установка

### NPX

```bash
npx -y git-mob-mcp-server
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "git-mob": {
      "command": "npx",
      "args": ["-y", "git-mob-mcp-server"]
    }
  }
}
```

### VS Code

```json
{
  "mcp": {
    "servers": {
      "git-mob": {
        "command": "npx",
        "args": ["-y", "git-mob-mcp-server"]
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `setup_git_mob` | Настраивает git-mob глобально для пользователя |
| `setup_git_mob_locally` | Настраивает git-mob локально для текущего репозитория, когда он переопределяет конфигурацию git core.hooksPath... |
| `add_team_member` | Добавляет нового участника команды, используя его ключ, имя и email |
| `delete_team_member` | Удаляет участника команды по его ключу |
| `list_team_members` | Выводит список всех участников команды, добавленных в Git Mob |
| `set_mob_session_coauthors` | Устанавливает активную парную или моб-сессию, указывая ключи участников команды для включения в качестве соавторов... |
| `clear_mob_session` | Очищает активную моб или парную сессию |
| `list_mob_session_coauthors` | Выводит список всех соавторов, включенных в текущую активную моб или парную сессию |
| `list_mob_session_coauthor_trailers` | Выводит список git Co-authored-by трейлеров для соавторов, включенных в текущую активную моб или парную сессию... |
| `get_git_mob_cli_help` | Отображает общую справку и информацию об использовании для Git Mob CLI |

## Возможности

- Настройка git mob CLI
- Добавление / удаление / список участников команды
- Выбор участников команды для парной или моб-сессии
- Автоматическое добавление Co-authored-by для соавторов в сообщения коммитов во время парного программирования или моб-сессий

## Примечания

Предварительные требования: Node.js версии 18 или выше, Git v2.32 или новее, и git-mob CLI приложение должно быть установлено. Запустите MCP инструмент setup_git_mob один раз для настройки git-mob перед использованием других инструментов. Для конфигурации VS Code вы можете добавить конфигурацию в .vscode/mcp.json в вашем рабочем пространстве, чтобы поделиться с другими.
