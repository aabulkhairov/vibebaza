MCP сервер для проведения комплексных исследований сложных тем, который помогает глубоко изучать вопросы, находить релевантные источники и генерировать структурированные отчёты с корректными цитатами.

## Установка

### Настройка Python

```bash
python setup.py
```

### Опубликованный сервер

```bash
uvx mcp-server-deep-research
```

### Сборка для разработки

```bash
uv sync
uv build
uv publish
```

## Конфигурация

### Разработка (неопубликованные серверы)

```json
{
  "mcpServers": {
    "mcp-server-deep-research": {
      "command": "uv",
      "args": [
        "--directory",
        "/Users/username/repos/mcp-server-application/mcp-server-deep-research",
        "run",
        "mcp-server-deep-research"
      ]
    }
  }
}
```

### Опубликованные серверы

```json
{
  "mcpServers": {
    "mcp-server-deep-research": {
      "command": "uvx",
      "args": [
        "mcp-server-deep-research"
      ]
    }
  }
}
```

## Возможности

- Детализация вопросов - расширяет и уточняет исследовательские вопросы, определяет ключевые термины и концепции
- Генерация подвопросов - создаёт сфокусированные подвопросы для всестороннего охвата темы
- Интеграция веб-поиска - использует встроенный веб-поиск Claude для целенаправленного поиска
- Анализ контента - оценивает качество информации, синтезирует находки из множественных источников
- Генерация отчётов - создаёт хорошо структурированные, комплексные отчёты с корректными цитатами
- Структурированный рабочий процесс исследования с систематическим подходом
- Выводы на основе доказательств со сбалансированными перспективами

## Примеры использования

```
Выберите шаблон промпта deep-research из MCP и начните исследование, предоставив исследовательский вопрос
```
