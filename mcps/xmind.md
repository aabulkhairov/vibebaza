Сервер Model Context Protocol для анализа и запросов к интеллект-картам XMind, предоставляющий возможности поиска, извлечения и анализа содержимого файлов XMind.

## Установка

### Smithery

```bash
npx -y @smithery/cli install @41px/mcp-xmind --client claude
```

### Ручная установка

```bash
npm install @modelcontextprotocol/sdk adm-zip zod
npm install --save-dev typescript @types/node
```

### Прямой запуск

```bash
node dist/index.js <allowed-directory> [additional-directories...]
```

### NPX продакшн

```bash
npx -y @41px/mcp-xmind /path/to/xmind/directory
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `read_xmind` | Парсинг и анализ файлов XMind, извлечение полной структуры интеллект-карт |
| `get_todo_tasks` | Извлечение и анализ TODO задач, включая контекст задач и иерархию |
| `list_xmind_directory` | Рекурсивное сканирование файлов XMind, фильтрация и организация результатов |
| `read_multiple_xmind_files` | Одновременная обработка нескольких файлов, сравнение и анализ между файлами |
| `search_xmind_files` | Поиск файлов по шаблонам имен, рекурсивное сканирование директорий |
| `extract_node` | Умный нечеткий поиск по путям, ранжированные результаты поиска, полное извлечение поддерева |
| `extract_node_by_id` | Прямой доступ к узлу по ID, быстрое и точное получение |
| `search_nodes` | Многокритериальный поиск по содержимому, настраиваемые поля поиска |

## Возможности

- Умный нечеткий поиск по интеллект-картам
- Управление задачами и отслеживание
- Навигация по иерархическому контенту
- Извлечение ссылок и референсов
- Анализ нескольких файлов
- Поддержка меток и тегов
- Сканирование директорий
- Безопасный доступ к директориям

## Примечания

Требует указания путей к директориям в качестве аргументов для безопасности. Поддерживает MCP Inspector для тестирования в разработке. Разрешает доступ только к указанным директориям с нормализацией и валидацией путей.
