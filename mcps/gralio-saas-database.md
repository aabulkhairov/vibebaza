Доступ к более чем 3 миллионам отзывов о SaaS-сервисах, детальным данным о ценах, альтернативам, рейтингам, информации о финансировании, метрикам роста, анализу настроений пользователей и функциональным данным о более чем 30 000 программных продуктов через MCP endpoint.

## Установка

### Тест Supergateway

```bash
npx -y supergateway --sse "https://market.gralio.ai/mcp/sse"
```

## Конфигурация

### Cursor

```json
{
    "mcpServers": {
        "gralio": {
            "url": "https://market.gralio.ai/mcp/sse"
        }
    }
}
```

### Windsurf & Cascade

```json
{
    "mcpServers": {
        "gralio": {
            "url": "https://market.gralio.ai/mcp/sse"
        }
    }
}
```

### VS Code

```json
{
    "servers": {
        "GralioSaaS": {
            "type": "sse",
            "url": "https://market.gralio.ai/mcp/sse"
        }
    }
}
```

### Claude Desktop

```json
{
    "mcpServers": {
        "gralio_remote": {
            "command": "npx",
            "args": [
                "-y",
                "supergateway",
                "--sse",
                "https://market.gralio.ai/mcp/sse"
            ]
        }
    }
}
```

## Возможности

- Более 3 млн отзывов о SaaS с глубокой обратной связью от пользователей и анализом настроений
- Детальные данные о ценах с полными тарифными планами и разбивкой по функциям
- Идентификация и сравнение альтернатив конкурентов
- Отслеживание финансирования и роста компаний с раундами инвестиций и траекториями роста
- Оценки настроений пользователей, собранные со всего интернета
- Информация о функциональных возможностях с детальными сведениями о возможностях продуктов
- Покрытие более 30 000 программных продуктов

## Примечания

Все данные доступны через бесплатный Gralio MCP endpoint по адресу https://market.gralio.ai/mcp/sse. Сервер использует протокол SSE (Server-Sent Events) и требует supergateway для интеграции с Claude Desktop.
