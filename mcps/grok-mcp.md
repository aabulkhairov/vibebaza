MCP сервер для xAI Grok API, предоставляющий доступ к возможностям включая понимание изображений, генерацию изображений, поиск в реальном времени и модели рассуждений.

## Установка

### Из исходного кода

```bash
git clone https://github.com/merterbak/Grok-MCP.git
cd Grok-MCP
uv venv
source .venv/bin/activate # macOS/Linux or .venv\Scripts\activate on Windows
uv sync
```

### Stdio

```bash
uv run python main.py
```

### Docker

```bash
docker compose up --build
```

### MCP Inspector

```bash
mcp dev main.py
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "grok": {
      "command": "uv",
      "args": [
        "--directory",
        "/path/to/Grok-MCP",
        "run",
        "python",
        "main.py"
      ],
      "env": {
        "XAI_API_KEY": "your_api_key_here"
      }
    }
  }
}
```

### Filesystem MCP (Опционально)

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/<your-username>/Desktop",
        "/Users/<your-username>/Downloads"
      ]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `list_models` | Список всех доступных моделей Grok с датами создания и информацией о владельце |
| `chat` | Стандартное завершение чата с обширными возможностями настройки |
| `chat_with_reasoning` | Получение детального рассуждения вместе с ответом |
| `chat_with_vision` | Анализ изображений с помощью естественных языковых запросов |
| `generate_image` | Создание изображений из текстовых описаний |
| `live_search` | Поиск в интернете в реальном времени с цитированием источников |
| `stateful_chat` | Поддержание состояния разговора через несколько запросов на серверах xAI |
| `retrieve_stateful_response` | Получение ранее сохраненного ответа разговора |
| `delete_stateful_response` | Удаление сохраненного разговора с серверов xAI |

## Возможности

- Множественные модели Grok: Доступ к Grok-4.1-Fast-Reasoning, Grok-4.1-Fast-Non-Reasoning, Grok-4-Fast, Grok-3-Mini и другим
- Генерация изображений: Создание изображений используя модель генерации изображений Grok
- Возможности компьютерного зрения: Анализ изображений с моделями зрения Grok
- Поиск в интернете в реальном времени: Поиск в реальном времени с цитатами из новостей, веб-страниц, X и RSS-лент
- Модели рассуждений: Продвинутые рассуждения с расширенными моделями мышления
- Состояние разговоров: Используйте эту недавно выпущенную функцию для поддержания контекста разговора как id через несколько запросов
- История разговоров: Включить или выключить для использования предыдущего контекста

## Примечания

Требует Python 3.11 или выше и Astral UV. Claude Desktop не может отправлять загруженные изображения напрямую в инструменты MCP, но вы можете использовать опциональный Filesystem MCP сервер для доступа к изображениям по пути к файлу. Поддерживает форматы изображений JPG, JPEG, PNG для возможностей компьютерного зрения. Состояние разговоров сохраняется на 30 дней на серверах xAI.
