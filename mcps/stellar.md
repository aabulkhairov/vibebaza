MCP сервер, который предоставляет возможности взаимодействия с блокчейном Stellar, позволяя LLM работать как со Stellar Classic, так и со смарт-контрактами Soroban, управлять аккаунтами и выполнять различные блокчейн операции.

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `stellar_create_account` | Создать новый Stellar аккаунт |
| `stellar_balance` | Получить баланс Stellar аккаунта |
| `stellar_payment` | Отправить платеж на другой аккаунт |
| `stellar_transactions` | Получить историю транзакций для аккаунта |
| `stellar_create_asset` | Создать новый актив в сети Stellar |
| `stellar_change_trust` | Изменить трастлайн для актива |
| `stellar_create_claimable_balance` | Создать запрашиваемый баланс, который может быть получен указанными аккаунтами при определенных условиях |
| `stellar_claim_claimable_balance` | Получить запрашиваемый баланс используя его ID |
| `stellar_fund_account` | Пополнить тестовый аккаунт используя Friendbot (только testnet) |
| `soroban_build_and_optimize` | Собрать и оптимизировать смарт-контракты Soroban |
| `soroban_deploy` | Развернуть смарт-контракты Soroban в сети Stellar |
| `soroban_retrieve_contract_methods` | Получить полный интерфейс развернутого смарт-контракта Soroban |

## Возможности

- Создание и управление Stellar аккаунтами
- Проверка балансов аккаунтов и истории транзакций
- Отправка платежей между аккаунтами
- Создание и управление пользовательскими активами
- Обработка трастлайнов для активов
- Создание и получение запрашиваемых балансов с условиями
- Пополнение testnet аккаунтов используя Friendbot
- Сборка и оптимизация смарт-контрактов Soroban
- Деплой смарт-контрактов с аргументами конструктора
- Получение полных интерфейсов контрактов включая методы, структуры и перечисления

## Примечания

Этот MCP сервер предоставляет комплексную функциональность блокчейна Stellar, включая как традиционные операции Stellar Classic, так и продвинутые возможности смарт-контрактов Soroban. Он поддерживает сложные типы данных, деплой контрактов с аргументами конструктора и детальную инспекцию интерфейсов контрактов.
