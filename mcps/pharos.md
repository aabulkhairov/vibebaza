MCP сервер, предоставляющий доступ к базе данных Pharos — комплексной базе знаний NIH для изучения лекарственного генома, белков, заболеваний и химических соединений в исследованиях по открытию лекарств.

## Установка

### NPX Remote

```bash
npx mcp-remote YOUR_PHAROS_MCP_SERVER_URL_HERE
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "pharos": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "YOUR_PHAROS_MCP_SERVER_URL_HERE"
      ]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `pharos_graphql_query` | Выполняет GraphQL запросы к Pharos API для получения информации о мишенях, заболеваниях, ... |

## Возможности

- Запросы мишеней (белков) с использованием идентификаторов, таких как UniProt ID
- Запросы заболеваний по названию и поиск связанных мишеней
- Запросы лигандов (соединений/лекарств) по идентификаторам, таким как ChEMBL ID
- Выполнение GraphQL интроспекции для понимания схемы Pharos API
- Доступ к комплексной информации о функциях белков и связанных заболеваниях
- Получение данных о химических соединениях, включая статус одобрения лекарств и биологическую активность

## Примеры использования

```
Pharos, расскажи мне о белке с UniProt ID P05067.
```

```
Используя Pharos, какие известные мишени есть для болезни Альцгеймера?
```

```
Можешь использовать Pharos для поиска информации о лиганде CHEMBL12?
```

## Примечания

Сервер разработан для развертывания на Cloudflare Workers и подключается к Pharos GraphQL эндпоинту по адресу https://pharos-api.ncats.io/graphql. Доступен под лицензией MIT с требованием академического цитирования. Формат URL сервера: https://pharos-mcp-server.your-account-name.workers.dev/sse
