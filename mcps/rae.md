MCP сервер, который предоставляет доступ к словарю и лингвистическим ресурсам Королевской академии испанского языка (RAE), позволяя языковым моделям искать и получать подробную информацию об испанском языке.

## Установка

### Из исходного кода

```bash
git clone https://github.com/rae-api-com/rae-mpc.git
cd rae-mpc
go build
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `search` | Поиск в RAE API информации с запросом и опциональными языковыми параметрами |
| `get_word_info` | Получение подробной информации о конкретном слове с опциональными языковыми параметрами |

## Возможности

- Поиск в словаре RAE и лингвистических ресурсах
- Получение подробной информации о словах и определениях
- Поддержка множества языков (по умолчанию испанский)
- Интеграция с stdio транспортом для LLM
- Поддержка режима SSE сервера

## Ресурсы

- [GitHub Repository](https://github.com/rae-api-com/rae-mcp)

## Примечания

Требует Go 1.21+. Может работать с stdio транспортом для интеграции с LLM или как SSE сервер на указанном порту. Язык по умолчанию - испанский (es), но поддерживаются и другие языковые коды.
