MCP сервер, который предоставляет AI-ассистентам прямой доступ к данным блокчейна Ergo через стандартизированный интерфейс, позволяя исследовать блокчейн, анализировать адреса, получать информацию о токенах и проводить продвинутую аналитику.

## Установка

### Из исходного кода

```bash
git clone https://github.com/ergo-mcp/ergo-explorer-mcp.git
cd ergo-explorer-mcp
pip install -r requirements.txt
python -m ergo_explorer.server
```

### Docker

```bash
docker build -t ergo-explorer-mcp .
docker run -d -p 8000:8000 \
  -e ERGO_EXPLORER_API="https://api.ergoplatform.com/api/v1" \
  -e ERGO_NODE_API="http://your-node-address:9053" \
  -e ERGO_NODE_API_KEY="your-api-key" \
  --name ergo-mcp ergo-explorer-mcp
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `openwebui_entity_tool` | Возвращает текстовое резюме обнаруженных сущностей для адреса |
| `openwebui_viz_tool` | Отображает интерактивную сетевую визуализацию D3.js для кластеризации адресов |

## Возможности

- Исследование блокчейна: Получение блоков, транзакций и сетевой статистики
- Анализ адресов: Запрос балансов, истории транзакций и проведение криминалистического анализа
- Аналитика токенов: Просмотр информации о токенах, распределения держателей, отслеживание исторического владения и данные коллекций
- Интеграция экосистемы: Доступ к информации EIP, данным пулов оракулов и адресной книге
- Продвинутая аналитика: Анализ паттернов блокчейна, метрик токенов и потоков транзакций
- Идентификация сущностей: Обнаружение связанных адресов с помощью продвинутых алгоритмов кластеризации адресов
- Интерактивные визуализации: Генерация и взаимодействие с сетевыми визуализациями для анализа сущностей
- Стандартизация ответов: Поддержка двух форматов (JSON/Markdown) с консистентной структурой
- Оценка токенов: Встроенный подсчет токенов для оптимизации контекстного окна AI
- Отслеживание исторических держателей токенов: Полная история токенов с отслеживанием высоты блока

## Переменные окружения

### Обязательные
- `ERGO_EXPLORER_API` - URL для Ergo Explorer API

### Опциональные
- `ERGO_NODE_API` - URL для Ergo Node API (опционально для продвинутых функций)
- `ERGO_NODE_API_KEY` - API ключ для Ergo Node API (опционально)

## Примечания

Сервер включает комплексные возможности идентификации сущностей через алгоритмы кластеризации адресов, интеграцию с Open WebUI для улучшенной визуализации и встроенную оценку токенов для оптимизации AI контекста. Все ответы следуют стандартизированному формату с метаданными, включающими время выполнения, размер результата и оценки токенов.
