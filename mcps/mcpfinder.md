MCPfinder позволяет языковым моделям искать и устанавливать новые возможности MCP сервера по требованию, работая как "App Store для агентов", который превращает статичные модели в развивающихся агентов, охотящихся за новыми возможностями.

## Установка

### Интерактивная настройка

```bash
npx -y @mcpfinder/server install
```

### Облачный доступ (HTTP/SSE)

```bash
claude mcp add --transport http mcpfinder https://mcpfinder.dev/mcp
```

### Из исходников

```bash
git clone https://github.com/mcpfinder/server
node index.js
```

## Конфигурация

### Stdio транспорт (локальный)

```json
{
  "mcpServers": {
    "mcpfinder": {
      "command": "npx",
      "args": [
        "-y",
        "@mcpfinder/server"
      ]
    }
  }
}
```

### HTTP/SSE транспорт (облачный)

```json
{
  "mcpServers": {
    "mcpfinder": {
      "url": "https://mcpfinder.dev/mcp",
      "transport": "http"
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `search_mcp_servers` | Ищет в реестре MCPfinder доступные MCP серверы по ключевым словам или тегам |
| `get_mcp_server_details` | Получает подробную информацию о конкретном MCP сервере, включая его манифест и установку... |
| `add_mcp_server_config` | Добавляет или обновляет конфигурацию MCP сервера в локальной конфигурации клиентского приложения... |
| `remove_mcp_server_config` | Удаляет конфигурацию конкретного MCP сервера из локальной конфигурации клиентского приложения... |

## Возможности

- Настройка без лишних усилий с мгновенной доступностью
- AI-готовое автономное обнаружение и установка инструментов
- Улучшение возможностей в реальном времени по требованию
- Поддержка как stdio, так и HTTP/SSE транспортов
- Интерактивный инструмент настройки для автоматической конфигурации MCP
- Реестр проверенных MCP серверов, не требующих ручной настройки
- Возможность регистрации сервера для издателей
- Поддержка как Node.js (npx), так и Python (uvx) пакетов

## Переменные окружения

### Опциональные
- `MCPFINDER_API_URL` - Базовый URL для API реестра MCPfinder
- `MCP_PORT` - Номер порта, на котором сервер будет слушать в режиме HTTP

## Примечания

Реестр содержит несколько сотен серверов, которые можно запускать локально с помощью npx или uvx без необходимости настройки переменных окружения. Сервер действует как основной интерфейс для поиска и включения новых инструментов, когда запрошенные возможности недоступны в текущем наборе инструментов MCP.
