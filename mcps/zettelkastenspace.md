AI-система для управления знаниями, основанная на методе Zettelkasten с интеграцией Model Context Protocol, которая позволяет Claude Desktop понимать и взаимодействовать с вашей персональной базой знаний.

## Возможности

- AI-сеть знаний с взаимосвязанными заметками
- Интеграция с Claude Desktop через Model Context Protocol
- Контекстно-зависимый AI, который понимает всю вашу базу знаний
- Семантический поиск и обнаружение за пределами простого поиска по ключевым словам
- Визуальный граф знаний и аналитическая панель
- Синхронизация в реальном времени между базой знаний и AI контекстом
- Умные теги и автоматические предложения связей
- Адаптивный дизайн для мобильных устройств
- Детальный контроль приватности и разрешений для данных
- Командная работа с общими базами знаний

## Примеры использования

```
Find connections between my AI ethics notes and governance research
```

```
Connect this new paper to my existing research
```

## Примечания

Это веб-приложение на zettelkasten.space, а не традиционная установка MCP сервера. Пользователи регистрируются через веб-интерфейс, а затем подключаются к Claude Desktop с помощью однокликовой настройки MCP. Система обеспечивает безопасную аутентификацию с ротацией токенов и позволяет пользователям контролировать, к каким данным AI может получить доступ через детальные разрешения.
