MCP сервер, который обеспечивает доступ к Foursquare Places API, предоставляя детальный контекст местоположения с базой данных из 100 миллионов мест в 1500+ категориях для создания ситуационно-осведомленных AI агентов.

## Возможности

- Доступ к Foursquare Places API с 100 миллионами мест в 1500+ категориях
- GeoTagging API с технологией Place Snap для высокоточной идентификации местоположения
- Search & Data API с фильтрацией по категориям, возможностям, часам работы и многому другому
- Богатые метаданные, включая фотографии, отзывы, рейтинги и популярность в реальном времени
- Преобразует необработанные GPS данные в значимые инсайты о местоположении
- Позволяет создавать ситуационно-осведомленные AI агенты для персонализированного опыта

## Переменные окружения

### Обязательные
- `FOURSQUARE_SERVICE_API_KEY` - Foursquare Service API ключ, необходимый для доступа к Foursquare API эндпоинтам

## Ресурсы

- [GitHub Repository](https://github.com/foursquare/foursquare-places-mcp)

## Примечания

Требует Claude Desktop App для локального использования MCP. Python реализация доступна в директории fsq-server-python с настройкой uv. Бесплатная учетная запись Foursquare разработчика включает стартовый кредит для проектов.
