MCPHub Desktop — это GUI приложение, которое служит установщиком, менеджером и каталогом для серверов Model Context Protocol (MCP), позволяя разработчикам без опыта работы с CLI устанавливать и управлять MCP серверами.

## Возможности

- Обнаружение зависимостей, включая node и uv(python)
- Установка зависимостей одним кликом
- Просмотр последних серверов из нашего сообщества
- Установка нужных серверов одним кликом
- Конфигурация и управление необходимыми переменными серверов через простой GUI
- Удаление ненужных серверов одним кликом
- Поддержка управления несколькими клиентами, такими как Claude
- Поддержка пользовательских аргументов командной строки, таких как пути к файлам и каталогам

## Ресурсы

- [GitHub Repository](https://github.com/Jeamee/MCPHub-Desktop)

## Примечания

Это настольное GUI приложение, а не сам MCP сервер. Оно управляет другими MCP серверами. Новые MCP серверы можно отправить через PR в MCPHub Servers Repo по адресу https://github.com/Jeamee/mcp-servers. Лицензировано под AGPL License.
