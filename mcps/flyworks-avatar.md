MCP сервер, который предоставляет бесплатное и быстрое создание видео с синхронизацией губ для цифровых аватаров без предварительного обучения, поддерживающий как реалистичные, так и мультяшные стили с возможностями text-to-speech и аудио ввода.

## Установка

### Smithery

```bash
npx -y @smithery/cli install @Flyworks-AI/flyworks-mcp --client claude
```

### Из исходного кода

```bash
git clone https://github.com/yourusername/flyworks-mcp.git
cd flyworks-mcp
pip install httpx "mcp[cli]>=1.6.0"
```

### Используя uv

```bash
uv pip install httpx "mcp[cli]>=1.6.0"
```

### Со всеми зависимостями

```bash
pip install pygments pydantic-core httpx "mcp[cli]>=1.6.0"
```

## Конфигурация

### Claude Desktop

```json
{
  "mcpServers": {
    "flyworks": {
      "command": "uvx",
      "args": [
        "flyworks-mcp",
        "-y"
      ],
      "env": {
        "FLYWORKS_API_TOKEN": "your_api_token_here",
        "FLYWORKS_API_BASE_URL": "https://hfw-api.hifly.cc/api/v2/hifly",
        "FLYWORKS_MCP_BASE_PATH": "/path/to/your/output/directory"
      }
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------|-------------|
| `create_lipsync_video_by_audio` | Создание видео с синхронизацией губ с аудио вводом, анимирующее цифрового аватара для речи в син... |
| `create_lipsync_video_by_text` | Создание видео с синхронизацией губ с текстовым вводом, генерирующее аудио из текста и анимирующ... |

## Возможности

- Создание видео с синхронизацией губ, используя видео цифрового аватара и аудио в качестве входных данных
- Создание видео с синхронизацией губ по тексту (с text-to-speech)
- Создание цифровых аватаров из изображений или видео
- Поддержка как асинхронного, так и синхронного режимов работы
- Поддержка реалистичных и мультяшных стилей аватаров
- Бесплатный пробный доступ с ограниченной квотой

## Переменные окружения

### Обязательные
- `FLYWORKS_API_TOKEN` - API токен для аутентификации в сервисе Flyworks

### Опциональные
- `FLYWORKS_API_BASE_URL` - Базовый URL для Flyworks API
- `FLYWORKS_MCP_BASE_PATH` - Путь к директории, где будут сохранены выходные файлы

## Примечания

Бесплатный пробный токен '2aeda3bcefac46a3' доступен с ограниченной дневной квотой, видео с водяными знаками и лимитом длительности 45 секунд. Создание аватаров через видео обеспечивает лучшее качество, но занимает больше времени, чем создание на основе изображений. Максимальное время ожидания для синхронного режима составляет 10 минут. URL видео временные и должны быть скачаны незамедлительно.
