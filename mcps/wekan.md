Неофициальный MCP сервер для Wekan, который предоставляет функциональность REST API для добавления, редактирования и удаления задач и досок в системе управления проектами Wekan.

## Возможности

- Добавление, редактирование и удаление задач и досок через REST API
- Автоматическая генерация API токенов
- Кроссплатформенные скрипты для генерации токенов (Windows PowerShell и Linux/macOS Bash)
- Управление конфигурацией окружения
- Возможности тестирования аутентификации
- Интеграция с MCP Inspector для разработки и отладки

## Переменные окружения

### Обязательные
- `WEKAN_BASE_URL` - URL вашего экземпляра Wekan
- `WEKAN_API_TOKEN` - Сгенерированный API токен для аутентификации
- `WEKAN_USER_ID` - Ваш ID пользователя Wekan
- `WEKAN_TOKEN_EXPIRES` - Дата истечения токена

## Примечания

Включает автоматизированные скрипты для генерации токенов, которые запрашивают эндпоинт Wekan, имя пользователя и пароль. Поддерживает рабочий процесс разработки с MCP Inspector и автоматическую пересборку. Доступна ручная конфигурация через настройку .env файла.
