A2AJava интегрирует A2A-MCP напрямую в Java-приложения. Позволяет разработчикам аннотировать обычные Java методы и мгновенно экспортировать их как MCP сервер, A2A-совместимые действия — без бойлерплейта и накладных расходов на регистрацию сервисов.
