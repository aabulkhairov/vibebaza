Комплексный сервер Model Context Protocol (MCP), который обеспечивает расширенную функциональность извлечения текста из PDF, поиска и анализа с настраиваемыми опциями для обработки локальных PDF файлов.

## Установка

### NPM Install

```bash
npm install
```

## Конфигурация

### Cursor

```json
{
  "mcpServers": {
    "mcp-gp-pdf-reader": {
      "command": "node",
      "args": ["/absolute/path/to/mcp_gp_pdf_reader/index.js"]
    }
  }
}
```

## Доступные инструменты

| Инструмент | Описание |
|------------|----------|
| `read-pdf` | Извлечение текста из PDF файлов с настраиваемыми опциями, включая диапазоны страниц, включение метаданных... |
| `search-pdf` | Поиск определенного текста в PDF документах с опциями поиска с учетом регистра, целых слов и regex... |
| `pdf-metadata` | Получение комплексных метаданных из PDF файлов, включая автора, заголовок, дату создания, ключевые слова и ф... |

## Возможности

- Извлечение текста из PDF файлов с настраиваемыми опциями
- Продвинутый поиск текста с опциями учета регистра, целых слов и regex
- Комплексное извлечение метаданных (автор, заголовок, дата создания, ключевые слова)
- Постраничная обработка и извлечение контента из определенных диапазонов страниц
- Возможности очистки и нормализации текста
- Защита ограничений размера файла (лимит 50MB)
- Асинхронная обработка для неблокирующих файловых операций
- Валидация файлов и санитизация путей для безопасности

## Примеры использования

```
Extract all text from /documents/report.pdf
```

```
Search for 'quarterly results' in /documents/financial-report.pdf
```

```
Get metadata from /documents/contract.pdf
```

## Примечания

Требует Node.js 18.0.0 или выше. Планируемые будущие функции включают поддержку OCR, извлечение изображений, определение таблиц, извлечение данных форм, поддержку защищенных паролем PDF и возможности пакетной обработки.
