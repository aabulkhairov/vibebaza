## Community Platform Options

| Platform | Best For | Considerations |
|----------|----------|----------------|
| Slack | B2B, engaged | Real-time, can be noisy |
| Discord | Tech, gaming | Feature-rich, learning curve |
| Circle | Courses, memberships | Paid, professional |
| Discourse | Tech, open source | Async, searchable |
| Facebook Groups | B2C, mainstream | Algorithm dependent |
| LinkedIn Groups | B2B professional | Limited features |
