## CRO Process

### ResearchXL Framework
1. **Technical Analysis:** Site speed, errors, mobile
2. **Heuristic Analysis:** UX expert review
3. **Analytics Analysis:** Funnel, drop-offs
4. **Mouse Tracking:** Heat maps, recordings
5. **Qualitative:** Surveys, user tests
6. **Prioritize:** Score opportunities

## A/B Testing Framework

### Hypothesis Structure
**Because** [observation]
**We believe** [change]
**Will result in** [outcome]
**We'll measure** [metric]

### Sample Size Calculation
- Baseline conversion rate
- Minimum detectable effect
- Statistical significance (95%)
- Statistical power (80%)
- Traffic volume

### Test Duration
- Minimum 2 weeks
- Full business cycles
- Reach statistical significance
- Account for external factors

## Key Metrics

| Metric | Formula |
|--------|---------|
| Conversion Rate | Conversions / Visitors |
| Bounce Rate | Single page visits / Total visits |
| Exit Rate | Exits from page / Visits to page |
| Click-Through Rate | Clicks / Impressions |
| Revenue Per Visitor | Revenue / Visitors |
