## Attribution Models

| Model | Description | Use Case |
|-------|-------------|----------|
| First Touch | Credit to first interaction | Awareness focus |
| Last Touch | Credit to last interaction | Conversion focus |
| Linear | Equal credit to all touches | Balanced view |
| Time Decay | More credit to recent touches | Long cycles |
| Position Based | 40/20/40 to first/middle/last | Common default |
| Data-Driven | Algorithm-based | Advanced teams |
