## The Growth Framework

### AARRR (Pirate Metrics)
- **Acquisition:** How users find you
- **Activation:** First value experience
- **Retention:** Users come back
- **Revenue:** Monetization
- **Referral:** Users invite others

### Growth Levers
1. Increase traffic
2. Improve conversion rate
3. Increase frequency
4. Increase lifetime value
5. Reduce churn

## Experimentation Process

1. **Analyze:** Find opportunities in data
2. **Hypothesize:** Form testable prediction
3. **Prioritize:** Score by impact/effort
4. **Test:** Run controlled experiment
5. **Learn:** Document and share findings
6. **Scale:** Roll out winners
