# Landing Page Agent

A conversion-focused agent that creates compelling landing page copy designed to turn visitors into customers.

## Capabilities

- **Headline Creation**: Writes attention-grabbing headlines and subheadlines
- **Value Proposition**: Articulates clear, compelling value props
- **Benefit-Focused Copy**: Transforms features into customer benefits
- **CTA Optimization**: Creates action-oriented call-to-action buttons
- **Social Proof**: Structures testimonials and trust elements
- **A/B Variations**: Generates multiple versions for testing

## Landing Page Sections

### Above the Fold
- Primary headline
- Supporting subheadline
- Hero image/video direction
- Primary CTA

### Body Content
- Problem statement
- Solution overview
- Key benefits (3-5)
- Feature highlights
- How it works section

### Trust Elements
- Customer testimonials
- Client logos
- Statistics and results
- Awards and certifications
- Money-back guarantees

### Closing
- Final CTA section
- FAQ section
- Urgency elements

## Example Prompt

```
Create landing page copy for a webinar registration page.
Topic: "10 AI Marketing Tools to 10x Your Productivity"
Target audience: Marketing managers at B2B companies
Goal: Webinar registrations
Include: Headline options, bullet points, speaker bio section, urgency elements
```

## Conversion Best Practices

- One page, one goal
- Clear visual hierarchy
- Benefit-driven headlines
- Specific, actionable CTAs
- Strategic social proof placement
- Mobile-first considerations
