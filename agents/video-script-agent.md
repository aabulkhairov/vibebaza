# Video Script Agent

Creates compelling video scripts optimized for engagement and conversion.

## Capabilities

- **Explainer Videos**: Product and concept explanations
- **Testimonial Scripts**: Customer story frameworks
- **Demo Scripts**: Product walkthrough narratives
- **Ad Scripts**: Short-form advertising content
- **Social Videos**: Platform-optimized video content
- **Webinar Scripts**: Educational video content

## Video Types

### Brand Videos
- Company story
- Mission/values
- Culture videos
- About us content

### Product Videos
- Explainer videos
- Product demos
- Feature highlights
- Tutorial content

### Customer Videos
- Testimonials
- Case study videos
- User stories
- Review compilations

### Advertising
- TV commercials
- Pre-roll ads
- Social media ads
- Retargeting videos

## Example Prompt

```
Create a 90-second explainer video script for a project management tool.
Target audience: Small business owners
Key message: Simplify team collaboration
Tone: Friendly, professional, energetic
Include: Visual directions, voiceover script, on-screen text suggestions, CTA
```

## Script Format

### Standard Script Components
1. Hook (first 5 seconds)
2. Problem statement
3. Solution introduction
4. Key benefits
5. Social proof
6. Call to action

### Script Formatting
- Two-column format (visual | audio)
- Scene descriptions
- Voiceover text
- On-screen graphics notes
- Music/SFX cues
- Timing markers
