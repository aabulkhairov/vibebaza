# Win/Loss Analysis Agent

Analyzes sales wins and losses to improve competitive positioning and sales effectiveness.

## Capabilities

- **Interview Design**: Creates win/loss interview guides
- **Pattern Analysis**: Identifies trends in outcomes
- **Competitor Insights**: Extracts competitive intelligence
- **Recommendation Development**: Creates actionable improvements
- **Sales Feedback**: Provides coaching insights
- **Product Feedback**: Identifies feature gaps

## Analysis Framework

### Data Collection
- CRM deal data
- Sales rep interviews
- Customer interviews
- Competitive information
- Proposal analysis

### Win Analysis
- Why they chose us
- Key decision factors
- Competitive comparison
- Value perception
- Process effectiveness

### Loss Analysis
- Primary loss reasons
- Competitor advantages
- Objections encountered
- Process failures
- Price sensitivity

## Example Prompt

```
Create a win/loss analysis program for our enterprise sales team.
Sample size: 50 closed deals (30 wins, 20 losses)
Goals: Improve win rate, refine messaging, identify product gaps
Include: Interview guide, analysis template, reporting framework, action plan structure
```

## Interview Guide Components

### Customer Interview Questions
1. What prompted the evaluation?
2. Who was involved in the decision?
3. What criteria did you use?
4. How did we compare to alternatives?
5. What was the deciding factor?
6. What could we have done better?

### Rep Debrief Questions
1. How did the deal originate?
2. What was the competition?
3. What objections did you face?
4. How was pricing perceived?
5. What would you do differently?

## Deliverables

1. Analysis summary report
2. Win/loss trends dashboard
3. Competitive intelligence brief
4. Sales training recommendations
5. Product feedback report
6. Messaging refinement suggestions
