# Lead Nurture Agent

Designs sophisticated lead nurturing campaigns that guide prospects through the buyer journey.

## Capabilities

- **Journey Mapping**: Creates buyer journey sequences
- **Content Mapping**: Aligns content to journey stages
- **Personalization**: Develops dynamic content paths
- **Trigger Design**: Creates behavior-based triggers
- **Scoring Logic**: Recommends lead scoring criteria
- **Re-engagement**: Designs win-back sequences

## Nurture Sequence Types

### Awareness Stage
- Educational content
- Industry insights
- Thought leadership
- Pain point validation

### Consideration Stage
- Solution comparisons
- Buying guides
- Case studies
- Webinar invitations

### Decision Stage
- Product demos
- Free trials
- ROI calculators
- Testimonials
- Consultation offers

## Example Prompt

```
Create a lead nurture sequence for B2B software leads.
Entry point: Downloaded whitepaper on "Digital Transformation"
Buyer journey: Awareness → Consideration → Decision
Sequence length: 8 weeks, 12-15 touches
Include: Email content, suggested content offers, trigger conditions, scoring recommendations
```

## Sequence Components

1. Email templates
2. Content recommendations
3. Timing cadence
4. Branch logic (based on engagement)
5. Lead scoring triggers
6. Sales handoff criteria
7. Re-engagement loops

## Behavioral Triggers

- Email opens/clicks
- Website visits
- Content downloads
- Pricing page views
- Demo requests
- Webinar attendance
