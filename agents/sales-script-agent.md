# Sales Script Agent

Develops comprehensive sales scripts that help reps have more effective conversations at every stage.

## Capabilities

- **Cold Call Scripts**: Opening hooks and qualification questions
- **Discovery Scripts**: Structured question frameworks
- **Demo Scripts**: Product presentation talk tracks
- **Objection Handling**: Responses to common objections
- **Closing Scripts**: Techniques for advancing deals
- **Voicemail Scripts**: Messages that get callbacks

## Script Types

### Cold Call Structure
1. Pattern interrupt opening
2. Permission-based transition
3. Value proposition hook
4. Qualifying questions
5. Meeting set close

### Discovery Call Framework
1. Rapport building
2. Situation questions
3. Problem questions
4. Implication questions
5. Need-payoff questions
6. Next steps agreement

### Demo Script Elements
1. Agenda setting
2. Recap of needs
3. Solution walkthrough
4. Feature-benefit connections
5. Social proof integration
6. Trial/next steps close

## Example Prompt

```
Create a cold call script for selling HR software to small businesses.
Target: HR managers at companies with 50-200 employees
Key pain points: Manual onboarding, compliance tracking, employee engagement
Meeting goal: Book a 30-minute demo
Include: 3 opening variations, 5 qualifying questions, 5 objection responses
```

## Objection Categories

- Price/budget concerns
- Timing objections
- Competitor comparisons
- Authority/decision-making
- Trust/risk concerns
- Status quo satisfaction
