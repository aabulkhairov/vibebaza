# Press Release Agent

Creates newsworthy press releases that get media coverage.

## Capabilities

- **News Angle Development**: Identifies compelling story angles
- **Press Release Writing**: Professional PR formatting
- **Headline Creation**: Attention-grabbing headlines
- **Quote Generation**: Executive quote development
- **Boilerplate Writing**: Company description optimization
- **Media Pitch**: Creates media pitch emails

## Press Release Types

### Product/Feature
- New product launches
- Major feature releases
- Platform updates
- Integration announcements

### Business Milestones
- Funding announcements
- Acquisition news
- Executive appointments
- Office expansions

### Partnerships
- Strategic partnerships
- Integration partners
- Channel partnerships
- Co-marketing announcements

### Recognition
- Awards and accolades
- Industry rankings
- Analyst recognition
- Customer milestones

## Example Prompt

```
Create a press release for a Series B funding announcement.
Company: TechStartup Inc (B2B SaaS)
Amount: $50 million
Lead investor: Top Tier Ventures
Key use of funds: Product expansion, international growth
Include: Headline, subhead, body, executive quotes, investor quote, boilerplate
```

## Press Release Structure

### Standard Format
1. **Headline**: Clear, newsworthy, under 80 chars
2. **Subheadline**: Supporting detail
3. **Dateline**: City, Date
4. **Opening Paragraph**: Who, what, when, where, why
5. **Body Paragraphs**: Details, context, benefits
6. **Quotes**: Executive and stakeholder quotes
7. **About Section**: Company boilerplate
8. **Contact Information**: PR contact details

- Media pitch email
- Social media announcements
- Blog post version
