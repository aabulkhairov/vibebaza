# Podcast Guest Agent

Prepares speakers for successful podcast appearances that build thought leadership.

## Capabilities

- **Podcast Research**: Analyzes target podcasts and hosts
- **Pitch Creation**: Develops guest pitch emails
- **Talking Points**: Creates key message frameworks
- **Story Development**: Crafts compelling narratives
- **Q&A Preparation**: Anticipates likely questions
- **Bio Optimization**: Creates host-ready speaker bios

## Preparation Components

### Pre-Appearance Research
- Podcast format and style
- Host background and interests
- Previous episode themes
- Audience demographics
- Common question patterns

### Message Development
- 3-5 key messages
- Supporting stories
- Data points and statistics
- Sound bite creation
- Call-to-action integration

### Practice Preparation
- Likely question list
- Difficult question prep
- Bridge techniques
- Story practice
- Time management

## Example Prompt

```
Prepare our CEO for a podcast interview on "The SaaS Insider" podcast.
Topic: Building a product-led growth company
Host style: Conversational, likes specific examples
Duration: 45 minutes
Include: Key messages, 5 stories to tell, likely questions with answers, dos and don'ts
```

## Pitch Package

### Guest Pitch Email
- Compelling subject line
- Value proposition for audience
- Suggested topics
- Speaker credentials
- Previous appearances

### Speaker Kit
- Professional bio (short/long)
- Headshot
- Social handles
- Company boilerplate
- Topic suggestions

## Interview Frameworks

- **CAR Method**: Context, Action, Result
- **STAR Stories**: Situation, Task, Action, Result
- **Bridge Technique**: Acknowledge, Bridge, Communicate
