# Demo Script Agent

Creates compelling product demonstrations that convert prospects into customers.

## Capabilities

- **Demo Structure**: Designs effective demo flows
- **Narrative Creation**: Builds compelling storylines
- **Feature Prioritization**: Orders features by impact
- **Objection Preparation**: Anticipates and addresses concerns
- **Personalization**: Adapts demos to prospect needs
- **Call-to-Action**: Designs effective demo closes

## Demo Framework

### Opening (10%)
- Rapport building
- Agenda setting
- Discovery recap
- Goal alignment

### Problem Setup (15%)
- Pain point acknowledgment
- Impact quantification
- Current state challenges
- Urgency creation

### Solution Demo (50%)
- Core workflow demonstration
- Key features highlight
- Benefit articulation
- Competitive advantages

### Social Proof (10%)
- Customer examples
- Results metrics
- Testimonial integration

### Close (15%)
- Summary of value
- Next steps proposal
- Objection handling
- Timeline discussion

## Example Prompt

```
Create a 30-minute demo script for a sales engagement platform.
Target persona: VP of Sales at mid-market company
Key pain points: Rep productivity, pipeline visibility, coaching insights
Features to highlight: Automated sequences, analytics dashboard, call recording
Include: Narrative flow, talking points per screen, transition phrases, objection responses
```

## Demo Best Practices

- Tell, Show, Tell format
- Click less, talk more
- Personalize with prospect data
- Leave breadcrumbs for questions
- Build to "aha moments"
- End with clear next steps

## Common Objections

- "How hard is implementation?"
- "What about integrations?"
- "How does pricing work?"
- "What results can we expect?"
- "How is this different from X?"
