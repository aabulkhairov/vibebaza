# Sales Proposal Agent

Generates professional, customized sales proposals that win deals.

## Capabilities

- **Custom Proposals**: Tailored to specific prospect needs
- **Executive Summaries**: Compelling opening overviews
- **Solution Design**: Technical approach documentation
- **ROI Calculations**: Quantified value justification
- **Pricing Presentation**: Strategic pricing structures
- **Implementation Plans**: Rollout timelines and milestones

## Proposal Structure

### Executive Summary
- Client situation understanding
- Proposed solution overview
- Expected outcomes
- Investment summary

### About Us
- Company overview
- Relevant experience
- Differentiators
- Team credentials

### Understanding Your Needs
- Current challenges
- Goals and objectives
- Success criteria
- Stakeholder requirements

### Proposed Solution
- Solution overview
- Key features and benefits
- Technical specifications
- Integration details

### Implementation
- Project approach
- Timeline and phases
- Resource requirements
- Risk mitigation

### Investment
- Pricing breakdown
- Payment terms
- ROI analysis
- TCO comparison

## Example Prompt

```
Create a sales proposal for enterprise CRM implementation.
Prospect: Global manufacturing company, 5000 employees
Needs: Unified customer view, sales automation, marketing integration
Deal size: $250K annually
Include: Executive summary, solution overview, implementation timeline, ROI analysis
```

## Customization Options

- Industry-specific language
- Compliance considerations
- Multi-stakeholder versions
- Competitive comparisons
- Custom ROI models
