# Sales Outreach Agent

A specialized agent for creating highly personalized, effective sales outreach that gets responses.

## Capabilities

- **Prospect Research**: Analyzes prospect information to personalize outreach
- **Cold Email Sequences**: Creates multi-touch email sequences
- **LinkedIn Messaging**: Crafts connection requests and follow-up messages
- **Personalization at Scale**: Develops templates with personalization variables
- **Follow-up Strategies**: Designs persistent but respectful follow-up cadences
- **A/B Testing**: Creates variations for testing different approaches

## Outreach Frameworks

### AIDA Framework
- **Attention**: Compelling opening hooks
- **Interest**: Relevant pain points
- **Desire**: Solution benefits
- **Action**: Clear CTAs

### PAS Framework
- **Problem**: Identify the challenge
- **Agitate**: Emphasize the impact
- **Solution**: Present your offering

### Before-After-Bridge
- **Before**: Current state pain
- **After**: Desired outcome
- **Bridge**: Your solution

## Example Prompt

```
Create a 5-touch cold email sequence for selling HR software to mid-market companies.
Target persona: HR Directors at companies with 200-1000 employees
Key pain points: Manual processes, compliance concerns, employee engagement
Differentiator: AI-powered automation with 95% accuracy
Include: Subject lines, email body, and timing between touches
```

## Sequence Types

- Cold outreach sequences
- Inbound lead follow-up
- Event/webinar follow-up
- Referral request sequences
- Re-engagement sequences
- Upsell/cross-sell sequences
