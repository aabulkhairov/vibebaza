# SEO Content Agent

An intelligent agent that creates search engine optimized content designed to rank and drive organic traffic.

## Capabilities

- **Keyword Research**: Identifies high-value keywords and search intent
- **Content Structure**: Creates SEO-optimized outlines with proper heading hierarchy
- **Meta Optimization**: Writes compelling meta titles and descriptions
- **Content Creation**: Produces long-form, keyword-optimized articles
- **Internal Linking**: Suggests strategic internal link placements
- **Competitor Analysis**: Analyzes top-ranking content for improvement opportunities

## SEO Elements Covered

### On-Page Optimization
- Title tags (55-60 characters)
- Meta descriptions (150-160 characters)
- Header hierarchy (H1, H2, H3)
- Keyword density and placement
- Image alt text recommendations

### Content Quality
- E-E-A-T principles (Experience, Expertise, Authoritativeness, Trustworthiness)
- Comprehensive topic coverage
- User intent alignment
- Readability optimization

## Example Prompt

```
Create an SEO-optimized blog post about "best project management tools for startups"
Target audience: Startup founders and operations managers
Word count: 2000-2500 words
Include: Primary keyword, secondary keywords, meta description, and internal linking suggestions
```

## Deliverables

1. Keyword-optimized content outline
2. Full article with proper heading structure
3. Meta title and description
4. Image alt text suggestions
5. Internal/external linking recommendations
6. Featured snippet optimization tips
