# Event Marketing Agent

Complete event marketing support from planning to post-event follow-up.

## Capabilities

- **Event Strategy**: Develops event marketing plans
- **Booth Content**: Creates trade show booth materials
- **Speaker Content**: Develops presentation content
- **Promotional Campaigns**: Pre-event marketing
- **Lead Capture**: Designs lead collection strategies
- **Follow-up Sequences**: Post-event nurture campaigns

## Event Types

### In-Person Events
- Trade shows and conferences
- User conferences
- Roadshows
- Executive dinners
- Networking events

### Virtual Events
- Virtual conferences
- Online workshops
- Virtual booth experiences
- Live streaming events

### Hybrid Events
- In-person + virtual attendance
- On-demand content libraries
- Networking lounges

## Example Prompt

```
Create an event marketing plan for exhibiting at a major industry conference.
Event: SaaS Industry Conference, 5000 attendees
Booth size: 10x10 standard
Goals: 150 qualified leads, 20 meetings booked
Include: Booth design concepts, pre-show outreach, at-event activities, post-event follow-up
```

## Pre-Event Marketing

- Email invitation sequence
- Social media campaign
- Meeting scheduler
- Booth appointment booking
- Teaser content
- Speaker promotion

## At-Event Execution

- Booth engagement activities
- Demo stations
- Lead capture methods
- Speaking sessions
- Networking events
- Real-time social posting

## Post-Event Follow-up

- Thank you emails
- Lead segmentation
- Demo scheduling
- Content delivery
- Survey distribution
- ROI calculation
