# Ad Copy Agent

A specialized agent for creating high-performing ad copy across all major advertising platforms.

## Capabilities

- **Google Ads**: Search ads, responsive ads, display ads
- **Meta Ads**: Facebook and Instagram ad copy
- **LinkedIn Ads**: Sponsored content and InMail
- **Twitter/X Ads**: Promoted tweets and campaigns
- **Platform Compliance**: Follows character limits and policies
- **A/B Variations**: Creates multiple versions for testing

## Platform Specifications

### Google Search Ads
- Headlines: Up to 15 (30 chars each)
- Descriptions: Up to 4 (90 chars each)
- Display URL paths

### Meta Ads
- Primary text: 125 chars visible
- Headline: 40 chars
- Description: 30 chars
- Various formats (carousel, video, collection)

### LinkedIn Ads
- Introductory text: 150 chars
- Headline: 70 chars
- Sponsored InMail subject: 60 chars

## Example Prompt

```
Create Google Search Ads for a cloud accounting software.
Target keywords: "accounting software for small business", "cloud bookkeeping"
USPs: Free trial, integrates with 200+ apps, AI-powered categorization
Create: 10 headlines, 4 descriptions, and 3 complete ad combinations
```

## Ad Copy Frameworks

- **Problem-Solution**: Address pain, offer solution
- **Social Proof**: Lead with results/testimonials
- **Urgency**: Time-limited offers
- **Curiosity**: Question-based hooks
- **Direct Benefit**: Lead with the outcome
