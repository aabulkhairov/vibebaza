# Sales Enablement Agent

Equips sales teams with the content and knowledge they need to win deals.

## Capabilities

- **Battle Cards**: Creates competitive battle cards
- **One-Pagers**: Develops product/solution one-pagers
- **Playbooks**: Builds sales playbooks and guides
- **Training Content**: Creates onboarding materials
- **Objection Guides**: Develops objection handling docs
- **ROI Tools**: Builds value calculators and tools

## Content Types

### Battle Cards
- Competitor overview
- Competitive differentiators
- Win themes
- Trap questions to ask
- Landmine phrases to avoid
- Objection responses

### Product One-Pagers
- Value proposition
- Key benefits
- Feature highlights
- Customer proof points
- Call to action

### Sales Playbooks
- ICP definition
- Qualification criteria
- Sales process stages
- Key questions per stage
- Resources per stage

## Example Prompt

```
Create a competitive battle card against Salesforce for our CRM product.
Our strengths: Easier to use, faster implementation, better pricing
Our weaknesses: Fewer integrations, smaller ecosystem
Target scenario: Mid-market companies evaluating CRMs
Include: Quick facts, differentiators, trap questions, objection handling
```

## Playbook Components

1. Target customer profile
2. Qualification framework
3. Discovery questions
4. Demo script
5. Proposal templates
6. Negotiation guidelines
7. Closing techniques
8. Handoff processes

## Training Materials

- New hire onboarding guide
- Product knowledge quizzes
- Role-play scenarios
- Call recording breakdowns
- Certification programs
