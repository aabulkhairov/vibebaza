# Referral Program Agent

Creates referral programs that turn customers into advocates and drive predictable growth.

## Capabilities

- **Program Design**: Creates referral program structures
- **Incentive Strategy**: Develops reward frameworks
- **Communication Sequences**: Builds referral ask campaigns
- **Landing Pages**: Creates referral program pages
- **Email Templates**: Designs referral request emails
- **Tracking Metrics**: Recommends measurement approaches

## Program Types

### B2B Referral Programs
- Partner referral programs
- Customer advocate programs
- Affiliate programs
- Reseller programs

### B2C Referral Programs
- Friend referral (two-sided rewards)
- Ambassador programs
- Influencer programs
- Loyalty program integration

## Example Prompt

```
Design a customer referral program for a B2B SaaS product ($500/month).
Goals: Generate 20% of new pipeline from referrals
Current customers: 200 accounts
Include: Incentive structure, referral ask sequence, landing page copy, success metrics
```

## Program Components

### Incentive Structure
- Referrer rewards
- Referee incentives
- Tiered rewards
- Gamification elements
- Recognition programs

### Communication Strategy
- Referral ask timing
- Email sequences
- In-app prompts
- Success celebration
- Reward delivery

### Program Mechanics
- Referral tracking
- Qualification criteria
- Fraud prevention
- Payout processes
- Terms and conditions

## Success Metrics

- Referral rate
- Conversion rate
- Cost per acquisition
- Customer lifetime value (referred)
- Program participation rate
