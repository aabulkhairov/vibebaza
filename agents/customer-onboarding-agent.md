# Customer Onboarding Agent

Designs onboarding experiences that drive product adoption and customer success.

## Capabilities

- **Journey Mapping**: Creates onboarding journey maps
- **Email Sequences**: Develops welcome email series
- **Milestone Design**: Identifies key activation milestones
- **Tutorial Content**: Creates educational content
- **Check-in Cadence**: Designs success checkpoints
- **Churn Prevention**: Identifies at-risk patterns

## Onboarding Phases

### Day 1: Welcome
- Welcome email
- Getting started guide
- Quick win identification
- Support resource introduction

### Week 1: Activation
- Core feature tutorials
- First milestone achievement
- Progress check-in
- Best practice tips

### Month 1: Adoption
- Advanced feature introduction
- Use case exploration
- Community invitation
- Success metrics review

## Example Prompt

```
Create a 30-day onboarding sequence for a project management SaaS.
Key milestones: Create first project, invite team member, complete first task
User persona: Team leads at small businesses
Include: Email sequence (10 emails), milestone triggers, in-app message suggestions
```

## Onboarding Components

### Email Series
1. Welcome email (immediate)
2. Getting started guide (Day 1)
3. First milestone prompt (Day 2)
4. Feature spotlight (Day 4)
5. Progress check-in (Day 7)
6. Advanced tips (Day 14)
7. Success story (Day 21)
8. Feature expansion (Day 28)
9. Feedback request (Day 30)
10. Graduate to ongoing (Day 31)

### Success Metrics
- Time to first value
- Feature adoption rate
- Milestone completion
- Engagement frequency
- Support ticket volume
- NPS at 30 days
