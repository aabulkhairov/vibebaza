# Customer Persona Agent

Creates comprehensive buyer personas to align marketing and sales efforts with customer needs.

## Capabilities

- **Demographic Profiling**: Age, location, income, job title analysis
- **Psychographic Analysis**: Values, interests, lifestyle mapping
- **Pain Point Discovery**: Identifies challenges and frustrations
- **Goal Mapping**: Understands aspirations and objectives
- **Buying Behavior**: Analyzes decision-making processes
- **Channel Preferences**: Determines preferred communication channels

## Persona Components

### Demographics
- Job title and seniority
- Company size and industry
- Location and timezone
- Income/budget range
- Education level

### Psychographics
- Values and beliefs
- Professional goals
- Personal interests
- Information sources
- Communication style

### Buying Behavior
- Decision-making process
- Key stakeholders involved
- Evaluation criteria
- Common objections
- Preferred buying channels

## Example Prompt

```
Create a detailed buyer persona for a B2B project management tool.
Target market: Tech startups with 10-50 employees
Information available: Product is $15/user/month, focuses on developer teams
Create: 2 distinct personas (decision maker and end user)
Include: Quote, day-in-the-life scenario, objection handling tips
```

## Deliverables

1. Persona profile document
2. Day-in-the-life narrative
3. Pain point priority matrix
4. Messaging recommendations
5. Channel strategy suggestions
6. Objection handling guide
